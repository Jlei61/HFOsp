# results/topic4_sef_hfo/snn_heterogeneity/figures

放电网络（SNN）异质性病理核机制实验（局部化设计 + 2×2 轴，spec `2026-06-08-sef-hfo-snn-heterogeneity-mechanism-design.md`）。外围标量安静、异质性只放病理核；核做成 2×2（平均门槛 {18,16} × 参差 {宽,窄}）：baseline / matched(纯方差) / mean_only(纯均值) / unmatched(合并)。读出是 firing-density / current-LFP proxy（**不是 LFP**），只验方向顺序。

### grid_overview.png
2×2 网格每格病理核对"核内同时活跃比例"的影响（核 − 基线）。**蓝=matched(纯方差)、橙=mean_only(纯均值)、红=unmatched(合并)**。**hatch 斜纹 = 核戳前自发点着**（其 d 值是状态切换粗指标、**非诱发同步**，review 1B）；黑边 = 戳在核上（非机制证据）。一眼看：**蓝（方差）全小且无 hatch（不点着）；橙/红（mean-down）全 hatch（点着）** = 点不点着由均值定、参差零影响。

### baseline_compare.png
四个代表的整体网络状态（**源空间群体率**，竖虚线 = 戳 @150ms）：matched 与基线几乎重合、单个干净**诱发**事件（戳后 ~185ms）；mean_only / unmatched 在**戳之前** ~55–80ms 就有早峰 = **自点火灶**（标题标 IGNITES pre-kick）。直接展示"mean-down 核戳前自己烧起来"。

### mechanism_<tag>.png（4 联图：异质性 + 传播 + 电极读出，复用参考图画法）

- **b 传播地图**（viridis，5–95 分位裁色）：神经元面按**事件起始时刻**上色 + **电极触点按到达时刻上色**（传播顺序在电极上也可见）。
- **c ∥ 杆读出**：per-contact 峰值依次扫（**斜的峰轨迹 = 读出方向**）。
- **d ⊥ 杆读出**：峰值对齐（竖直 = 无方向）。

### sweep_ignition.png（多种子+位置 sweep：点火概率，2026-06-09）
7 个核位置（沿轴 far→near + 离轴）× 12 种子，三条线 = 三种病理改法的"戳前自点火比例"。**蓝（纯方差）贴 0（0/84，从不点着）；橙（压低平均+宽参差）贴 1（84/84 全点着）；红（压低平均+窄参差）大多 1、偶尔掉到 0.83（77/84）**。一眼看：**平均下移是点火主门；宽参差的低门槛尾巴帮点火（84/84），收窄略降（77/84）**——参差调"可靠性"、不是主开关。**修正早先"参差零作用"的过硬说法。**

### sweep_matched_evoked.png（纯方差核对诱发同步：无一致方向，2026-06-09）
只有纯方差核（matched，平均不动、从不点着）的诱发同步改变量，按位置画 mean±SEM。**跨位置正负摆动（−0.18 → +0.22），整体 pooled mean −0.023±0.026 ≈ 0**。标题写"no consistent direction"——诚实读法是"**跨位置没有稳定同向效应**"，**不是**"零效应"；位置依赖的摆动本身要看见。

**关注点**：ignition 图看"平均主门 + 参差调可靠性（84/84 vs 77/84）"；matched-evoked 图看"正负摆动、pooled≈0 但非稳定零"。自点火格子的 post-kick 传播/同步数只作状态指标、不进诱发结论（降级纪律）。

### mean_scan.png（平均门槛幅度扫：点火边界，2026-06-09）
3 联图，每联一个问题（§7）。把核平均门槛**一档一档下扫** 18→17.5→17→16.5→16，宽/窄参差各一条，**6 张独立网络 × 2 阈值场/噪声 = 每格 12 个独立实现**（连接骨架不再固定——补上 sweep 的"单网络" caveat）。
- **a 点火边界（headline）**：点火概率 vs 平均门槛。**平均=18 那档圈出 = sanity 锚，实测 0.00，精确复现 sweep 零结果**（无 setup 漂移）。18/17.5 都平地 0，点火在 17→16.5→16 段**逐档爬升** = **真实点火边界**（不是"16 一压就烧的太强档"）。**宽参差曲线整条在更高平均处**（橙在红左/上），边界橙 16.86mV vs 红 16.29mV，**水平位移 +0.57mV = 宽参差低门槛尾巴把点火边界往高平均推**（尾巴在边界附近帮点火）。竖点线 = 各自 0.5 交叉点。
- **b 点火潜伏期**（只算点着的 trial）：越往下压平均，点火越早越猛（宽 122→34ms）；同一平均下宽比窄点得更早（尾巴先引燃）。
- **c 边界以上的诱发同步**（只算没点着的 trial）：平均从 18 往边界走，非点火核的"一起放电"**逐步升高**（18 处≈0、贴 sweep 零结果；逼近边界核"热起来"）。**注意**：宽线在 16.5 以下因几乎全点着、非点火样本 n→1/0 而消失；宽/窄此处是**被选择的非点火子集、描述性**，干净的参差对比看 a 不看 c。

**关注点**：a 看"有没有真边界 + 宽尾把边界推多高（+0.57mV）"；b 看越界后点火多猛；c 看边界以上诱发同步爬升（18 处≈0）。诚实边界：6 网络=小样本独立网络（screen-plus，非正式裁决）；+0.57 位移基于 N=12 插值交叉、suggestive，需 16.5–17 一档加密二次确认。

### spontaneous_probe.png（B 侧旁支 = tail-driven nucleation，独立机制）
不戳：均匀标量安静（2 Hz）；宽参差铺满整片自发爆发（328 Hz @65ms）。**宽**门槛分布的低门槛尾细胞自点着造有限尺寸成核点。**方向纪律**：这是"变宽也能造起火点"，跟主线"变窄危险"不是同一命题，**不并进**"失去异质性"链。放电版看得见、率均值场看不见。**证据等级（2026-06-09 review）**：目前是**单次实现**，只能写成"**风险/强烈警报**"，**不是已坐实的机制**——升级需多种子 + 有限尺寸扫描 + 密度扫描排除单实现假象。

**关注点**：宽尾 → 自发成核（**风险线索，待多种子/有限尺寸/密度补强**），是局部化设计的动机。
