# results/topic4_sef_hfo/snn_heterogeneity/figures

放电网络（SNN）异质性病理核机制实验（局部化设计 + 2×2 轴，spec `2026-06-08-sef-hfo-snn-heterogeneity-mechanism-design.md`）。外围标量安静、异质性只放病理核；核做成 2×2（平均门槛 {18,16} × 参差 {宽,窄}）：baseline / matched(纯方差) / mean_only(纯均值) / unmatched(合并)。读出是 firing-density / current-LFP proxy（**不是 LFP**），只验方向顺序。

### grid_overview.png
2×2 网格每格病理核对"核内同时活跃比例"的影响（核 − 基线）。**蓝=matched(纯方差)、橙=mean_only(纯均值)、红=unmatched(合并)**。**hatch 斜纹 = 核戳前自发点着**（其 d 值是状态切换粗指标、**非诱发同步**，review 1B）；黑边 = 戳在核上（非机制证据）。一眼看：**蓝（方差）全小且无 hatch（不点着）；橙/红（mean-down）全 hatch（点着）** = 点不点着由均值定、参差零影响。

### baseline_compare.png
四个代表的整体网络状态（**源空间群体率**，竖虚线 = 戳 @150ms）：matched 与基线几乎重合、单个干净**诱发**事件（戳后 ~185ms）；mean_only / unmatched 在**戳之前** ~55–80ms 就有早峰 = **自点火灶**（标题标 IGNITES pre-kick）。直接展示"mean-down 核戳前自己烧起来"。

### mechanism_<tag>.png（4 联图：异质性 + 传播 + 电极读出，**每个参数组合一张**，2026-06-09 扩成全组合）

- **b 传播地图**（viridis，5–95 分位裁色）：神经元面按**事件起始时刻**上色 + **电极触点按到达时刻上色**（传播顺序在电极上也可见）。

- **d ⊥ 杆读出**：峰值对齐（竖直 = 无方向）。

**命名 = `{kick位置}kick_{核位置}core_{条件}`**，每个参数组合一张（共 **24 个**，来自 grid_metrics.json 全部 cell）。**覆盖范围（2026-06-09 扩成 matched 全因子）**：

- **mean_only / unmatched 条件 = 只在 end kick × 4 核（8）**：这两个条件自点火、传播面板只作图示，所以没跑它们的全因子（kick 位置对自点火格子的传播无意义）。
- 合计 16 + 8 = 24。（更全的 mean_only/unmatched × 其他 kick 没做——自点火使其传播图示价值低。）
- 标题带 **⚠ SELF-IGNITES pre-kick — b/c/d propagation ILLUSTRATIVE ONLY**（mean_only/unmatched 自点火格子，传播面板只作图示、不进结论，§2 降级纪律）或 **(evoked event)**（matched 干净诱发）。

**关注点**：a 看异质性在哪、b 看传播顺序、c 峰轨迹斜率 = 方向、d 对齐 = 无方向；a 边界环是均值跳变伪迹（看核内）；带 ⚠ 的传播面板别当结论。

### sweep_ignition.png（多种子+位置 sweep：点火概率，2026-06-09）
7 个核位置（沿轴 far→near + 离轴）× 12 种子，三条线 = 三种病理改法的"戳前自点火比例"。**蓝（纯方差）贴 0（0/84，从不点着）；橙（压低平均+宽参差）贴 1（84/84 全点着）；红（压低平均+窄参差）大多 1、偶尔掉到 0.83（77/84）**。一眼看：**平均下移是点火主门；宽参差的低门槛尾巴帮点火（84/84），收窄略降（77/84）**——参差调"可靠性"、不是主开关。**修正早先"参差零作用"的过硬说法。**

### sweep_matched_evoked.png（纯方差核对诱发同步：无一致方向，2026-06-09）
只有纯方差核（matched，平均不动、从不点着）的诱发同步改变量，按位置画 mean±SEM。**跨位置正负摆动（−0.18 → +0.22），整体 pooled mean −0.023±0.026 ≈ 0**。标题写"no consistent direction"——诚实读法是"**跨位置没有稳定同向效应**"，**不是**"零效应"；位置依赖的摆动本身要看见。

**关注点**：ignition 图看"平均主门 + 参差调可靠性（84/84 vs 77/84）"；matched-evoked 图看"正负摆动、pooled≈0 但非稳定零"。自点火格子的 post-kick 传播/同步数只作状态指标、不进诱发结论（降级纪律）。

### mean_scan.png（平均门槛幅度扫：点火边界，2026-06-09）
3 联图，每联一个问题（§7）。把核平均门槛**一档一档下扫** 18→17.5→17→16.5→16，宽/窄参差各一条，**6 张独立网络 × 2 阈值场/噪声 = 每格 12 个独立实现**（连接骨架不再固定——补上 sweep 的"单网络" caveat）。
- **a 点火边界（headline）**：点火概率 vs 平均门槛。**平均=18 那档圈出 = sanity 锚，实测 0.00，精确复现 sweep 零结果**（无 setup 漂移）。18/17.5 都平地 0，点火在 17→16.5→16 段**逐档爬升** = **真实点火边界**（不是"16 一压就烧的太强档"）。**宽参差曲线整条在更高平均处**（橙在红左/上）——load-bearing 证据是 **16.5 档纵向落差：宽 11/12 vs 窄 3/12**（原始分数，不报 p）；水平边界位移 +0.57mV（橙 16.86 vs 红 16.29）**只作汇总、suggestive**（两插值之差、真实不确定度 ~±0.3–0.4mV，别当精确值）。竖点线 = 各自 0.5 交叉点。
- **b 点火潜伏期**（只算点着的 trial）：越往下压平均，点火越早越猛（宽 122→34ms）；同一平均下宽比窄点得更早（尾巴先引燃）。
- **c 边界以上的诱发同步**：**窄参差到 16.5 才点着，所以它 18/17.5/17 档是完整 n=12 样本——一条干净的亚阈"核随平均下移而升温"曲线（+0.046→+0.064→+0.135，小但真实）**。宽参差因 17 档就开始点着、非点火样本 n→8/1，**宽这条才是被选择的"幸运子集"、只作描述**；同档宽/窄诱发对比别用。

**关注点**：a 看"有没有真边界 + 16.5 档 11/12 vs 3/12 的纵向落差"；b 看越界后点火多猛；c 看**窄参差**的亚阈升温曲线（18 处≈0、是干净的）。诚实边界：6 网络=小样本独立网络（screen-plus，非正式裁决）；+0.57 位移 suggestive，需 16.5–17 一档加密二次确认。

**关注点**：和 coarse 对照看——边界没动、宽仍在窄左、窄升温曲线同型 = 独立复制。

### kick_amp.png（kick 幅度稳健性，2026-06-09 自主探索 B）
固定 mid 核 + end kick，KICK_BOOST {1×,2×,3×}·ν_θ，只扫 matched+baseline（igniting 跨幅度 bit-identical）。
- **a matched 诱发同步 vs 幅度**：**2×（锁定）+0.030、3×（更强）+0.045 = 贴零 → matched-null 对更强刺激稳健**（roadmap P1 问题 = YES）；但 **1×（弱）= −0.406±0.086（10/12 网络为负）= 弱戳下收窄参差让核招募更少**（宽尾巴在刺激边缘帮招募，同点火尾巴机制、anti-Rich 方向）。
- **b sanity**：baseline 任何幅度都不自点火（0/0/0）；pre-kick 不变量 True（unmatched 潜伏期 56.4ms 三幅度一致）；whole-net 诱发峰 0.34–0.43（弱戳反而略高 = 弱戳更挑门槛分布）。

**关注点**：a 看 2×/3× 贴零（null 稳健）+ 1× 负（弱戳尾巴效应，新发现、非 overturn）；b 看 baseline 不自点火 + pre-kick 不变量。**不是 advisor 守的 overturn（更强刺激冒效应）**——locked-kick null 站得稳，1× 是弱-刺激 regime 细化。

### core_size.png（核尺寸轴，2026-06-09 自主探索 C，guarded）
病灶核半径 {0.3,0.45,0.6}mm（395/920/1628 细胞）× 平均 {18,17,16.5,16} × {宽,窄} × 6 网络。
- **a 点火概率 vs 平均，3 个核尺寸**（宽实线/窄虚线，颜色=半径）：核越大曲线越往高平均方向移 = **大核在更健康的平均就自点火**。mean=18 宽圈出 = 守护（必须≈0）。
- **b 点火边界 vs 核尺寸**：宽边界随尺寸上移 16.5→17.25、窄 16.0→16.62，宽 > 窄（尾巴专属）。守护 CLEAN（baseline 任何尺寸都不自点火）。
**结果**：更大的核→边界往高平均推（越大越危险）；**r=0.3（395 细胞）全程不点 = 临界核尺寸**（小核撑不起自点火，有限尺寸效应）；宽>窄每个尺寸都成立。**守护 L=3 通过**（advisor 警告的大核自爆只在 L=1 发生、L=3 安全）。

**关注点**：a 看大核曲线左移（高平均点火）+ mean=18 守护≈0；b 看边界随尺寸单调上移 + 宽>窄。核尺寸 × 尾巴宽度 = 低门槛细胞绝对数，决定能否集体自点火。

### spontaneous_probe.png（B 侧旁支 = tail-driven nucleation，独立机制）
不戳：均匀标量安静（2 Hz）；宽参差铺满整片自发爆发（328 Hz @65ms）。**宽**门槛分布的低门槛尾细胞自点着造有限尺寸成核点。**方向纪律**：这是"变宽也能造起火点"，跟主线"变窄危险"不是同一命题，**不并进**"失去异质性"链。放电版看得见、率均值场看不见。**证据等级（2026-06-09 review）**：目前是**单次实现**，只能写成"**风险/强烈警报**"，**不是已坐实的机制**——升级需多种子 + 有限尺寸扫描 + 密度扫描排除单实现假象。

**关注点**：宽尾 → 自发成核（**风险线索，待多种子/有限尺寸/密度补强**），是局部化设计的动机。
