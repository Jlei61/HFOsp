# results/topic4_sef_hfo/snn_heterogeneity/figures

放电网络（SNN）异质性病理核机制实验（局部化设计，spec `2026-06-08-sef-hfo-snn-heterogeneity-mechanism-design.md`）。外围标量安静、异质性只放病理核；同一戳点比"健康宽参差核 vs 病理窄核"。所有读出是 firing-density（**不是 LFP**），只验方向顺序。

### grid_overview.png
病理核在 (戳点 × 核配置) 粗网格上对"核内细胞同时活跃比例"的影响（核 − 基线）。**配平核（matc，只把参差收窄、平均不动）所有柱子都贴近 0** = 几乎无效应；**不配平核（unma，参差收窄 + 平均压低 = 更易激）柱子大且正负取决于核摆哪**（mid +0.54 / far −0.57）。红柱 = 戳在核上，按非循环纪律**不作机制证据**。

**关注点**：matched 一排都接近 0（参差本身无效应）vs unmatched 一排幅度大（效应来自"更易激"那一半）——这是本实验的主结论形状。

### baseline_compare.png
四个代表案例的整体网络状态：基线（宽参差健康核，灰）vs 病理核（红）的群体读出轨迹。**mid_matched 两条几乎重合**（单个干净诱发事件 ~185ms，收窄参差不改变）；**mid_unmatched 红线多出一个 ~80ms 的早峰**（更易激核在戳之前就自己点着）；maxeffect（far unmatched）同样早点着但把主波打散；nearzero（far matched）与基线重合。

**关注点**：matched 与基线重合 = null；unmatched 的早峰 = 更易激核自发点着（局部成核热点）。

### mechanism_mid_matched.png（最干净的机制图）
配平核（只收窄参差）。左 a：神经元面按邻域门槛离散度上色（核内 = 收窄后的低离散度区，外围标量 = 0），叠病理核虚线轮廓 + 戳点 + 两根电极杆 + 长轴箭头。中 b：∥ 杆 per-contact 峰值**依次扫过**（斜的峰轨迹 = 读出方向）；右 c：⊥ 杆峰值**对齐**（竖直 = 无方向）。展示安静外围 + 干净诱发传播，配平核几乎不扰动它。

**关注点**：这是"模型事件 → 电极观测 → 方向"整条链可见的范例；配平核读出与基线一致 = 参差收窄不改变传播。

### mechanism_mid_unmatched.png / mechanism_maxeffect.png / mechanism_nearzero.png
代表案例（不配平 mid / 不配平 far 极端 / 配平 far 近零）。**注意**：不配平核左 a 面板核边界有一圈高离散度，是核内外**平均门槛跳变**（16 vs 18）造成的伪环，不是核内真实参差——核内部才是相关区域；不配平核读出面板偏弱，因其事件被早自发点着打成两段（见 baseline_compare）。

**关注点**：不配平核左 a 的边界环是平均跳变伪迹（看核内部）；读出弱因事件两段化。

### spontaneous_probe.png（B 侧旁支发现）
不戳（无刺激）下：均匀（标量）网络安静（2 Hz）；宽参差铺满整片则**自发一阵阵爆发**（328 Hz @65ms）。宽参差的低门槛左尾细胞自点着，把安静可激点搅成自发活跃——这正是放电版看得见、而"抹平平均"的率均值场看不见（它把有限尺寸同步平均掉了）。这也是为什么主实验把异质性**局限在核里、外围保持安静**。

**关注点**：宽参差 → 自发爆发（放电版独有的有限尺寸失稳），是局部化设计的动机。
