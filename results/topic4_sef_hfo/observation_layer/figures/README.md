## 双电极读出（论文级模型图，一模型一张，不画总图）

同一套虚拟电极 + 同一个画图函数（`src/sef_hfo_plot.py::two_electrode_readout`）画两遍：
一根电极平行于 E→E 连接长轴、一根垂直；两个模型都把事件种子放在一端，事件单向传过去。
每张图三栏：**A** 几何（事件 + 两根电极 + 种子）；**B** 平行电极每个触点录到的信号波形（按位置叠放）；
**C** 垂直电极的波形。核心读法看 B/C 里把各触点波峰连起来的黑虚线（"波峰轨迹"）：平行电极
**斜着走**（波依次扫过各触点 = 读出了方向），垂直电极**竖直**（各触点几乎同时到 = 读不出方向）。

三处口径约定（2026-06-08 用户校正）：① B/C 灰带 = **这根电极把事件录全的时间跨度**（第一个到最后一个
参与触点的波峰），不是"场事件窗"——场事件窗（整片像素过阈的比例）比波扫过整根电极的时间**窄**，
若把波峰限制在场窗内，最先/最后被扫到的触点会被钉在窗边，斜线尾巴变假；改用真实到达后斜线全真。
② A 的 colorbar 是**相对时间**（0 = 踢了之后第一个激活），不是绝对仿真时刻。③ A 的电极标注：率场触点稀疏
（4mm 间距）逐个标；SNN 触点密（亚毫米、两杆中心触点重合），只标每根杆的两端（P0/P8、Q0/Q8），
中间触点按杆上顺序对应 B/C 的波形叠放顺序。连接轴只画箭头、不写"connectivity axis"文字。

### two_electrode_readout_rate.png

率场（连续场、24mm、真实 4mm 触点间距）。A：每像素**激活后相对时间**（颜色，紫早黄晚）——E→E 兴奋
连接沿 45° 长轴的传播范围是垂直向的 2 倍（各向同性短程抑制），所以激活被漏斗到这条长轴、成一条
对角带（"只有长轴被点亮"=连接长轴被显形，是机制本身不是 bug）；两根电极叠加（实心 = 录到事件、
空心 = 没被波扫到）。B：平行电极各触点录到的发放率密度波形叠放，波峰轨迹斜着走 = 行波依次扫过
（P0 在种子端最先 ~19ms、P5 在远端最后 ~145ms；内部 P1→P3 是 33→70→105ms 的干净扫过）。
C：垂直电极各触点波峰对齐成竖线 = 同时到、读不出方向。率场是均场量，所以这里的"信号"是
**发放率密度，不是电流型 LFP**（图中已标 NOT a current）。

**关注点**：B 的波峰黑虚线是否斜、C 是否竖；波峰是否落在灰带内部（不再钉在边缘）；A 里电极是否压在
传播带上、colorbar 是否相对时间；信号标注是否写明"非电流"。

### two_electrode_readout_snn.png

脉冲网络（3mm 小片，会点火；触点按片缩放到亚毫米）。A：每个神经元**发放后相对时间**（颜色），
两根电极只标四个杆端（P0/P8、Q0/Q8；两杆中心触点重合，全标会糊），红星 = 种子。B/C：用**正式电流型
LFP**前向模型（|I_E|+|I_I|，在 ~2mm 小尺度验证过）录每个触点随时间的电信号。平行电极波峰轨迹斜着走
（峰时随位置单调移动 ~4ms = 行波扫过：**离种子最近的 P8 最先峰（184.8ms），最远的 P0 最后峰
（188.7ms）**——用户问的"O8 最早"=标注糊了误读的 P8，因为它最贴种子），垂直电极同跨度的触点波峰
几乎对齐（跨度 ~0.5ms = 同时到）。平行 ~4ms vs 垂直 ~0.5ms 的 8 倍反差才是"读出方向"的证据，不是
单看 P8 近。（幅度反而非单调：最近的 P8 峰最小，时序才是干净的读出。）

**关注点**：B 斜 / C 竖的对比、∥ 与 ⊥ 峰时跨度的反差；电信号是**电流型 LFP**（不是发放率密度）；
colorbar 是相对时间；杆端标注是否清晰；触点亚毫米、**非真实 SEEG 间距**（机制示意）；事件在片中心点火、
电极放在朝 [0,0] 的那条单向传播臂上；真实 4mm 尺度的定量读出由 rate 图承担，SNN 在 cm 尺度仍点不着。
