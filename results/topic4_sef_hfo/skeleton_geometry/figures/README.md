# 间期传播几何骨架 — 图说明

### axis_frame_examples.png

六个被试（primary 档、distributed 多杆采样）的传播骨架散点：横轴=离源沿轴距离(mm)，纵轴=离轴垂直距离(mm)，颜色=该通道的时序刻板性"超出随机的量"（raw excess = 实测 − 配对随机基线均值；蓝=低于随机、红=高于随机，色标对称居中于 0）。三角=源 core、方块=汇 core、圆=内部通道；灰线是从源(沿轴=0)到汇(沿轴=L)的传播轴。

**关注点**：看一个传播骨架在每个被试里"长什么样"——点沿横轴铺开=沿传播方向延伸，沿纵轴铺开=偏离轴的横向铺展（distributed 采样才能测到这个宽度）。颜色用的是 n-无关的 raw excess（不是会随事件数膨胀的 z），所以不同通道之间可直接比颜色深浅。

### skeleton_scalars_by_dataset.png

四联散点，**按数据集分层**：源-汇轴长、源 core 半径RMS、汇 core 半径RMS、参与通道垂向铺展RMS。每点一个被试；Yuquan(native mm)与 Epilepsiae(MNI-space mm)分两列、不混。**轴长那一联（最左）里，起止 core 沿轴交错（`sink_min_along < source_max_along`）的被试用红色空心圈标出**（图例 "weak axis (interleaved cores)"，4/26，全在 Epilepsiae 列）——这些被试源/汇重心相互抵消，轴长被弱约束、不能读成"模板紧凑"。垂向铺展那一联只画"可测"（非单杆 1D 采样）的被试。

**关注点**：Epilepsiae 坐标是 MNI 模板空间 mm（每个头被缩放到模板），与 Yuquan 真解剖 native mm **不能直接比绝对数值**——只看各数据集内部的尺度量级与分布。"垂向铺展"是**参与通道的垂向铺展**，受采样+发放率共同圈住，是一个**下界**，不是"斑块真实宽度"；单杆 1D 被试垂宽不可测、已被排除。

### along_axis_stereotypy_profile.png

沿传播轴的时序刻板剖面（pooled 所有 primary+fallback 被试）：横轴=离源沿轴距离(mm)，纵轴=刻板性 raw excess（实测 − 随机基线均值），虚线=随机水平(0)。

**关注点**：沿轴方向的刻板性**部分是模板定义出来的**（通道的 rank 本身≈它的沿轴位置），所以这张图是延展性/边界 sanity（看刻板模式沿轴能延多远、端点是不是硬边界），**不是一个独立检验**；真正新信息的"离轴耦合宽度"留到下一轮。纵轴是 n-无关的 raw excess，不是会被高发放率通道虚高的 z，所以跨被试 pool 是干净的。
