## 0. 计科学合同：要测的是什么

朴素话三段式（CLAUDE.md §8）：

- **测了什么**：把每个录制切成 30 分钟的小段，看 **传播 endpoint 通道集合**（哪些通道**最先点火**，不是哪些通道**经常被激活**）在不同 epoch 间漂得多厉害；同时看这些 endpoint 通道在脑里**占多大空间**（centroid RMS / mean pairwise / min enclosing ball）漂不漂；最后对有 swap 信号的 subject，看**核心病理区的通道数量**（decision-k）在不同 epoch 间是否变化。SEF-ITP 区分性预测：rate ≫ geometry，rank-based endpoint identity 稳，空间半径稳，decision-k 稳（病理空间结构不进展）。
- **怎么测的**（原 draft, **2026-05-23 calibration-fail 后 SUPERSEDED**：原"对每 cluster 算 masked mean rank"被 21/23 calibration mismatch 推翻，refactor 后改用 PR-6 同一 estimator `_legacy_hist_mean_rank` + `extract_endpoint_middle` per-cluster valid_mask；并且原"9 subject 子集还要 per-epoch 重跑 swap_sweep"也被 user-review v2 catch 推翻 — 所有 23 subject 都跑, swap_class 作 stratification context, 不作 subject-level gate。详见下方 §1.0 + §1.4）：per epoch 把该 epoch 内事件按 cluster 分组，对每 cluster 算 **template_rank** (`_legacy_hist_mean_rank` + argsort(argsort), PR-6 同 estimator), top-k 通道 = source, bottom-k 通道 = sink。然后算 source / sink 各自 spatial radius 指标 (centroid RMS, mean pairwise, min enclosing ball — k ≤ 3 完整, k ≥ 4 不完整, see Phase 3 caveat in framework §3.4 v1.0.7), 跟 global PR-6 anchoring 算 Jaccard。decision-k drift 对所有 23 subject 都跑, by swap_class 分层报告。
- **揭示什么**（pre-registered）：
  - **PASS**: rate Wilcoxon p<0.05 + Cohen's d ≥ 0.3 vs rank-endpoint Jaccard drift；spatial radius drift 同方向；decision-k 稳。
  - **NULL**: rate / geom drift 同尺度。
  - **FAIL**: geom > rate drift 显著反向。

### 1.0 Canonical helper — `_per_cluster_template_rank(ranks, bools, cluster_evt_idx)` — shared B1 + B3

> **2026-05-23 user-review correction** (was: `_masked_mean_rank` with arithmetic mean):
> Calibration with PR-6 anchoring requires the **same template estimator PR-6 uses**
> (`_legacy_hist_mean_rank` 3-bin modal histogram from `src.interictal_propagation`),
> not a simple masked arithmetic mean. The simple-mean version produced 21/23 mismatch
> on cohort calibration before refactor (different statistic = can't reproduce PR-6).
> After refactor with the canonical helper below + per-cluster valid_mask: 0/23 mismatch.

```python
def _per_cluster_template_rank(
    ranks: np.ndarray,
    bools: np.ndarray,
    cluster_evt_idx: np.ndarray,
) -> Tuple[np.ndarray, np.ndarray]:
    """Per-cluster template rank using PR-2's legacy estimator (same as PR-6 consumes).

    Returns (template_rank, valid_mask) per channel:
      - template_rank: argsort(argsort(_legacy_hist_mean_rank(masked_ranks))) — int 0..n_ch-1.
        Same estimator PR-2 stores in adaptive_cluster.clusters[k].template_rank.
        `_legacy_hist_mean_rank` already filters by `bools[ci] > 0` per channel AND drops
        NaN via `vals = vals[np.isfinite(vals)]` (NaN-safe by construction).
      - valid_mask: per-channel "any participation in cluster's epoch events"
        — mirrors PR-6 anchoring runner's `per_cluster_masks` (cluster_bools.any(axis=1)).
    """
    n_ch = ranks.shape[0]
    if cluster_evt_idx.size == 0:
        return np.zeros(n_ch, dtype=int), np.zeros(n_ch, dtype=bool)
    cl_ranks = ranks[:, cluster_evt_idx]
    cl_bools = bools[:, cluster_evt_idx]
    template = _legacy_hist_mean_rank(cl_ranks, cl_bools)
    template_rank = np.argsort(np.argsort(template, kind="stable"), kind="stable")
    valid_mask = cl_bools.any(axis=1)
    return template_rank, valid_mask
```

(Old `_masked_mean_rank` removed 2026-05-23 — was dead code after refactor; had a latent
NaN bug `cl_ranks * cl_bools` would propagate NaN for missing-block padding. The canonical
helper above goes through `_legacy_hist_mean_rank` which is NaN-safe.)

### 1.2 `compute_endpoint_spatial_radius(endpoint_indices, coords)` — B2

| # | 不变量 | 测试方法 |
|---|---|---|
| C7 | coords 单位 = mm（assert at boundary，Phase 1 已 lock） | TDD test: assert raises on cm input |
| C8 | k=1 → centroid_rms=0, mean_pairwise=NaN, min_enclosing_radius=0（degenerate cases 明确） | TDD test: 单点输入 |
| C9 | n=3 points forming equilateral triangle (边长 d) → centroid_rms = d/sqrt(3), mean_pairwise = d, min_enclosing_radius = d/sqrt(3) | TDD test: 已知几何 |
| C10 | min_enclosing_ball — 用 Welzl 不切实际 (k≤5)；用 brute force on subsets of 2 or 3 points 即可 | TDD test: 共线 → min_enclosing = half max pairwise distance |

**实现模板**：

```python
def compute_endpoint_spatial_radius(
    endpoint_indices: List[int],
    coords: np.ndarray,  # (n_ch, 3) mm
) -> Dict[str, float]:
    pts = coords[endpoint_indices]
    k = len(pts)
    if k == 0:
        return {"centroid_rms": float("nan"), "mean_pairwise": float("nan"), "min_enclosing_radius": float("nan")}
    if k == 1:
        return {"centroid_rms": 0.0, "mean_pairwise": float("nan"), "min_enclosing_radius": 0.0}
    centroid = pts.mean(axis=0)
    centroid_rms = float(np.sqrt(((pts - centroid) ** 2).sum(axis=1).mean()))
    pdist = np.linalg.norm(pts[:, None, :] - pts[None, :, :], axis=-1)
    iu = np.triu_indices(k, k=1)
    mean_pairwise = float(pdist[iu].mean())
    # Welzl 太重；min_enclosing_ball brute force on small k (≤ 5)
    min_enclosing_radius = _min_enclosing_ball_radius(pts)
    return {
        "centroid_rms": centroid_rms,
        "mean_pairwise": mean_pairwise,
        "min_enclosing_radius": min_enclosing_radius,
        "n_points": int(k),
    }


def _min_enclosing_ball_radius(pts: np.ndarray) -> float:
    """For k ≤ 5: try (a) sphere through 2 antipodal points; (b) circumsphere of all triples;
    return min radius that contains all points (within numeric tolerance)."""
    k = pts.shape[0]
    if k <= 1:
        return 0.0
    candidates = []
    # (a) 2-point spheres
    for i in range(k):
        for j in range(i + 1, k):
            c = 0.5 * (pts[i] + pts[j])
            r = 0.5 * float(np.linalg.norm(pts[i] - pts[j]))
            if _all_within(c, r, pts):
                candidates.append(r)
    # (b) circumsphere of 3 points (in 3D = circle in plane, sphere passes through those 3)
    for i in range(k):
        for j in range(i + 1, k):
            for l in range(j + 1, k):
                try:
                    c, r = _circumsphere_three_points(pts[i], pts[j], pts[l])
                    if _all_within(c, r, pts):
                        candidates.append(r)
                except np.linalg.LinAlgError:
                    continue
    if not candidates:
        # fallback: use farthest-point heuristic
        c = pts.mean(axis=0)
        return float(np.max(np.linalg.norm(pts - c, axis=1)))
    return min(candidates)
```

### 1.4 `compute_decision_k_drift(per_epoch_ranks_a, per_epoch_ranks_b, per_epoch_valid_a, per_epoch_valid_b, n_perm=500)` — B3

每 epoch 调 `compute_swap_score_sweep` 取 `decision_k` (or `None` if `swap_class != "strict"` per epoch — 与 cohort-level swap_class 用同一定义)。返回 `decision_k_per_epoch + std(decision_k) + cohort-style null`。

| # | 不变量 | 测试 |
|---|---|---|
| C11 | per-epoch rank_a/b 必须来自 **`_per_cluster_template_rank`**（与 B1 同 canonical helper, 内部调 `_legacy_hist_mean_rank` PR-6 同 estimator）, 不是 raw lagPatRank, 也不是简单 arithmetic mean (后者是 2026-05-23 早 draft, calibration fail 后已 SUPERSEDED) | TDD test: phantom 通道在 rank vector 中应该被 valid_mask 排除 |
| C12 | epoch 内 cluster events < `min_events_per_cluster=20` → drop epoch（不 carry-forward） | TDD test: 给 cluster A 只有 5 events → 该 epoch 输出 decision_k=None |
| C13 | swap_sweep `n_perm=500` for perf budget (advisor catch 4: full n_perm=1000 太贵) | code review |

**Runner-level gating correction (user-return v2 catch 2026-05-23)**: do **NOT** filter
at runner level by `swap_class in {strict, candidate}`. Compute drift for **all 23 subjects**
(swap_class as context only; summarizer stratifies). For swap-positive subjects drift is
informative; for swap=none drift is the noise/control baseline. Filtering at runner level
over-dilutes the cohort and prevents within-cohort baseline comparison.

实现概要（注意：`cluster_a` / `cluster_b` IDs **必须** 从 `primary_pair` JSON 传入，**不能** 用 `np.unique(labels)`，advisor catch (3) — 显式记录测的是哪一对 cluster）：

```python
def compute_decision_k_drift(
    ranks: np.ndarray,
    bools: np.ndarray,
    labels: np.ndarray,
    epochs: List[Dict],
    cluster_a: int,             # from primary_pair JSON
    cluster_b: int,             # from primary_pair JSON
    min_events_per_cluster: int = 20,
    n_perm: int = 500,
    seed: int = 0,
) -> Dict[str, Any]:
    decision_k_per_epoch = []
    for ei, ep in enumerate(epochs):
        evt_idx = ep["event_indices"]
        idx_a = evt_idx[labels[evt_idx] == cluster_a]
        idx_b = evt_idx[labels[evt_idx] == cluster_b]
        if len(idx_a) < min_events_per_cluster or len(idx_b) < min_events_per_cluster:
            decision_k_per_epoch.append(None)   # don't carry-forward (advisor catch 4)
            continue
        # Use the canonical template estimator (same as B1 calibration-passing helper)
        rank_a, valid_a = _per_cluster_template_rank(ranks, bools, idx_a)
        rank_b, valid_b = _per_cluster_template_rank(ranks, bools, idx_b)
        sweep = compute_swap_score_sweep(rank_a.astype(float), rank_b.astype(float),
                                          valid_a, valid_b,
                                          n_perm=n_perm, seed=seed + ei)
        decision_k_per_epoch.append(sweep.get("decision_k"))
    finite_k = [k for k in decision_k_per_epoch if k is not None]
    return {
        "decision_k_per_epoch": decision_k_per_epoch,
        "n_epochs_with_decision_k": len(finite_k),
        "decision_k_std": float(np.std(finite_k)) if len(finite_k) >= 2 else float("nan"),
        "decision_k_mean": float(np.mean(finite_k)) if finite_k else float("nan"),
        "decision_k_range": [int(min(finite_k)), int(max(finite_k))] if finite_k else None,
    }
```
