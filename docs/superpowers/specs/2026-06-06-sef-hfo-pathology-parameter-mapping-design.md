# SEF-HFO 病理 → 中观 → 参数 映射 + 首轮调参 Design Spec

> **状态**：draft 2026-06-06。本 spec 在 `docs/topic4_sef_itp_framework.md` v0.2 之上，**细化"当我们说这个模型是癫痫模型时，应当改变什么参数 / 引入什么机制"这一规范问题**，并给出第一轮可执行的调参计划（连接性 E→E 核 + E 细胞阈值异质性），LIF rate field 与 SNN 两版并行。
>
> **关系**：
> - **细化**（不替换）framework §2 核心方程的 framing、§7.3"具体机制多样、中观动力学收敛"的文献地位、§6.5 Phase 4 实施路线。
> - **保留**（不触碰）framework 已锁的 H1–H6 真实数据验收合同、cohort tier、phantom-rank 纪律、clinical SOZ 不作拟合标签的红线、v0.2 核心边界（只解释间期 HFO 群体事件的中观传播组织，不解释 clinical seizure onset，不解释 HFO 80–250 Hz carrier）。
> - **正式化** framework 红线 §8.4（低异质性不能直接等同易激性，必须经 transfer→gain→stability→finite-pulse 闭合）：本 spec 的"病理 → 中观读数 → 闭环/有限扰动"强制链就是 §8.4 的可执行版。
> - **不冲突** framework 红线 §8.1（HFO carrier 80–250 Hz 不分 E/I）：本 spec 的带符号 E/I 耦合是**群体率 / event-envelope 层**的动力学对象，**不**是对 HFO carrier 振荡来源的 E/I 归因 —— 见 §8 红线核对。
>
> **文献锚（已查证 2026-06-06）**：
> - Rich et al. 2022, *Cell Reports* 39(8):110863 —— 人脑致痫皮层 L5 锥体细胞生物物理异质性下降（膜片钳实测），模型中异质性下降使回路更易突然同步、跳进高发放态。→ 异质性轴的实测锚。
> - Huberfeld et al. 2007, *J Neurosci* 27(37):9866 —— 人 TLE 海马下托 >20% 锥体细胞 GABA 反转电位去极化（KCC2 缺失），布美他尼可压制间期活动。→ GABA / Cl⁻ 符号轴的实测锚（本轮不进主线，记为预注册预测）。
> - Lepeu et al. 2024, *Nat Commun* 15:6945 —— 单脉冲扰动量化韧性（能扛多大扰动不发作），韧性随兴奋升高而降。→ 有限扰动安全余量 = 序参量的锚；支持用"韧性/余量"而非"临界慢化"。
> - 支撑（framework §7.3 已收）：Baud et al. 2018 *Nat Commun* 9:88（IEA 昼夜 + 多日节律，发作集中在多日节律上升相 → 慢状态可外源，不必事件驱动）；Cressman 2009 / Jirsa 2014 Epileptor / Proix 2018。

## 0. 一句话承诺（朴素话）

我们要把"这块组织是癫痫组织"这件事，从一句口号变成一条**可计算、可证伪的链**：病理不直接等于"更兴奋"，它先改一批**底层物理旋钮**（细胞阈值分布、连接方向、耦合符号、慢状态），这些旋钮再改变一块组织"输入→放电"曲线的**斜率、弯曲度、各单元齐不齐**，这些再决定"小扰动会不会被放大、有限大的一推刹不刹得住"，最后才表现成我们数据里看得见的东西（事件多不多、多长、传多远、模板稳不稳、会不会失控）。

第一轮只碰两件跟数据最贴的事：**连接方向**（决定"往哪传"——对应正反两套模板共享一条轴）和 **E 细胞阈值分布**（决定"哪里容易点着"——对应 Rich 的异质性下降）。其它机制先不进主线。每个旋钮的效果方向**必须在锁定工作点上算出来，不预设**。

## 1. 核心纪律：强制链（spec 的硬规则）

> **病理机制 ≠ gain 上升。** 任一病理机制必须走完整条链，方向在工作点上算、**永不预设**：
>
> ```text
> 底层参数            →  中观读数                       →  闭环 / 有限扰动        →  可观测现象
> (细胞参数 /            (有效输入-输出曲线的:               (闭环稳定性:              (事件率 / 时长 /
>  耦合符号 /             一阶斜率 G,                       小扰动放大?              招募范围 /
>  连接几何 /             二阶曲率 H,                       有限扰动安全余量 S:        模板稳定性 /
>  慢状态)                跨单元离散度 Var,                  有限一推刹得住?)          失控风险)
>                         E/I 失配 Cov)
> ```
>
> **禁止**在 spec / archive / 任何 plan 直接写"机制 X 导致癫痫 / 机制 X 使 gain 上升"。必须写成"机制 X 改变 \<曲线的哪个性质\>，方向在工作点上计算，再经闭环稳定性与有限扰动余量决定是否更易激"。

链上每个箭头的**符号是计算输出，不是输入**。把所有箭头一次性算出来的那个计算，就是敏感度矩阵 `∂(可观测)/∂(底层参数)` —— 它同时是后续辨识性 gating 分析的引擎（§7.3）。

## 2. 中观读数层（桥接层定义）

中观这一层是病理（生物侧）和模型参数（仿真侧）**共同映射到的公共货币**。它包含**两套正交的"一阶/二阶"** + 两个闭环读数。全部是**读数，不是旋钮**。

**A. 沿输入方向求导（单条有效曲线的形状）**

```text
斜率(增益)  G_a = ∂Φ_a/∂h  | h=h0     # 输入多一点，放电多多少
曲率        H_a = ∂²Φ_a/∂h² | h=h0     # 再多推一点，灵敏度本身变高(>0,越推越猛)还是变低(<0,越推越钝)
```

**B. 跨单元做统计（一块 patch 里很多小单元的分布）**

```text
一阶: 平均增益   <G_E>, <G_I>
二阶: 离散/失配  Var(G_E), Var(G_I), Cov(G_E,G_I), Var(J_EI^eff), Cov(G_E,J_EI^eff)
```

**C. 两个闭环读数**

```text
闭环放大   = max_k Re λ_max[ D_τ^{-1} (-I + G·Ŵ(k)) ]   # 小扰动会不会被放大(离失稳多近)；诊断量
安全余量   S = A_runaway - A_event                      # 有限一推刹不刹得住(离失控多近)；最终闸门
```

> 注：闭环放大（线性稳定性）只给地图；自限传播事件的**闸门是有限脉冲响应 A_event / A_runaway**。真 LIF 工作点是**稳健稳定但非线性可激**（framework 2026-06-03 数学路线更正），所以"近临界"降级为可能工作区，统一用韧性 / 余量 S 表述（Lepeu 锚）。

## 4. 病理 → 中观 → 参数 映射表（方向不预设）

每个箭头只说它**动了曲线的哪个性质**；符号留到工作点上算。

| 病理机制（朴素） | 先改的底层参数 | 改中观曲线的哪个性质（**方向 = 工作点上算**） | 文献锚 |
|---|---|---|---|
| **神经元异质性下降** | 收窄 patch 内 `p(V_th,E)` 等分布宽度（尤其 `Var(V_th,E)`） | 同时改**斜率 + 曲率**（变陡 / 变钝看工作点在上升段还是饱和段）+ **跨单元离散度↓** | Rich 2022 |
| **E 更易兴奋** | `V_th,E↓` / `E_L,E↑` / 背景 `I_E↑` | 移动工作点 → 改**平均发放率 + 局部斜率**（增益升降都可能）；更易激与否看闭环 + 余量 | — |
| **去抑制 ① I→E 弱** | `J_EI` 幅度↓ | 去掉 E 上的**直接负反馈** | — |
| **去抑制 ② E→I 弱** | `J_IE↓` | I 得到的驱动变少 → **间接**去抑制 + 改 E-I 环**时序** | — |
| **去抑制 ③ I 细胞减少** | `N_I` / I 连接密度↓ | 抑制**总量↓** + 抑制的**空间覆盖**缩（surround 弱） | — |
| **GABA 去极化** | `E_GABA` 上移 | I→E 有效作用从**超极化 / 分流抑制 → 去极化推动**；**是否真变正反馈取决于 `E_GABA` 相对工作膜电位 + 电导状态（分流仍可净抑制）** | Huberfeld 2007 |
| **慢状态漂移** | 慢变量 `q(t)` 缓调 `V_th / I_E / J / E_GABA` | 沿时间**移动工作点** → 余量随时间变，传播几何不变 | Baud 2018 / Epileptor |

**移出主表（纪律）**：
- **背景噪声 `σ`**：是**状态 / 外源输入，不是病理旋钮**（既可平滑曲线降峰值斜率，也可让低阈下细胞更易发放，方向不干净）。除非有文献锚，不进"癫痫病理方向"主表；作为状态变量保留。
- **连接几何（`ℓ_∥, ℓ_⊥, θ_EE`）**：**独立轴，不是细胞增益的一阶/二阶统计**；但它经连接谱 `Ŵ(k)` **参与闭环稳定性计算** → 决定扰动**往哪传**。是"独立对象 + 共同进稳定性"，不是"跟增益无关"。

## 5. 第一轮调参计划（两条最贴数据的轴）

**一句话**：**连接性先定"往哪传"，异质性先定"哪里容易点着"。** 其它机制先不进主线。

### 5.0 调参合同（4 条，2026-06-06 user review 加固 —— 写在最前，§5.1/§5.2/§6 必须遵守）

> 这四条防的是同一类病：以为在扫"几何 / 分布形状"，其实顺带改了"总强度 / 平均工作点"，于是把两个效应混在一起读。

1. **扫连接几何时固定总 E→E mass / 入度。** `W_EE` 是**归一化核**：扫 `θ_EE / ρ / ℓ` 时保持总积分 `∫W_EE dx'`（或总入度）**固定**；`w_EE` 是**唯一**的总强度 / 工作点 / 余量旋钮，单独扫。理由：2D 高斯核积分 ∝ `J_EE·ℓ_∥·ℓ_⊥`，直接改 `ℓ/ρ` 会顺带改核面积 = 顺带改总 E→E 回授强度，"测往哪传"时实际也在改"多容易点着"。
2. **扫异质性时加 mean-matched control。** 收窄 `Var(V_th,E)` 会因曲线凸性（Jensen）改变平均发放率。每个异质性实验**至少两层**：(a) raw perturbation；(b) **mean-rate-matched 或 mean-drive-matched control**（重定均值，使收窄方差**不**顺带移动平均工作点）。这样才能分开"分布变窄本身"与"平均工作点移动"。
3. **异质核是否成为下一步，取决于 `joint_A_failmode`（不预先写死）。** 承接 2026-06-04 Step-1 同质率场 NULL，但走向**条件式**：失败模式指向 **too-frequent / diffuse nucleation** → 进入异质核首轮；指向 **non-returning / long plateau** → **必须先补 recovery / surround**，或至少把异质核与 recovery/surround 并列 sensitivity。
4. **SNN 同构必须锁四件事。** ① 固定**期望入度** vs 固定**总 E→E 电流**（择一锁死）；② 改变 `ρ` 时**重归一化**权重；③ 收窄 `Var(V_th,E)` 时**保持均值**（配合合同 2）；④ 虚拟 SEEG **必须走真实模板提取 pipeline**（与真实数据同一套）。

### 5.1 连接性首轮 —— 只碰 E→E 空间连接核

理由：直接对应数据里最强的现象 —— **两类互逆、稳定的传播模板共享一条空间轴**。

```text
W_EE(x,x') = w_EE · N(θ_EE,ρ,ℓ) · exp[ -u²/(2ℓ_∥²) - v²/(2ℓ_⊥²) ]   # u 沿长轴, v 沿横向
归一化 N 使 ∫W_EE dx' = w_EE（总入度/总 mass 固定）；扫几何只改形状，不改总回授 —— 合同 1
```

| 参数 | 数据对应 | 角色 |
|---|---|---|
| `θ_EE`（E→E 长轴方向） | 正反模板的共同空间轴、通道先后顺序方向 | **最关键** |
| `ℓ_∥`（沿传播轴连接尺度） | 事件能跨多远、通道 lag 展布宽度 | 主 |
| `ℓ_⊥` 或 `ρ = ℓ_⊥/ℓ_∥`（横向 / 各向异性强度） | 模板方向是否稳定、传播是否被压成一条轴；`ρ→1` 接近无方向，`ρ` 太小则过度一维化 | 主 |
| `J_EE / w_EE`（E→E 总强度） | 工作点 + 有限扰动余量的一部分 | **不是方向的万能旋钮**；**禁止**"看不出事件就随手加它" |

**本轮不碰抑制连接**。`I→E surround` 是**自限机制的第二层**参数，不是第一层数据锚。第一层只问：**连接轴能不能决定往哪传，而不是电极杆决定往哪传。**

### 5.2 异质性首轮 —— 只做 E 细胞阈值分布的空间变化

理由：最贴 Rich 机制，也最容易在 LIF rate field 与 SNN 里**同构**实现。

| 参数 | 数据对应 / 实现 | 纪律 |
|---|---|---|
| `Var(V_th,E)` inside patch | LIF：用更窄的 `p(V_th,E)` 重积分得有效曲线；SNN：直接让 patch 内 E 神经元 V_th 分布变窄 | **方向不预设**，必须算它对斜率 / 曲率 / 闭环稳定 / 有限扰动余量的影响 |
| `x_patch`（病理核位置） | 事件最常从哪成核、source 端是否偏向此区 | **不要拟合成 clinical SOZ**；它先是模型里的成核热点 |
| `r_patch`（病理核空间大小） | 参与通道范围、source/sink 空间半径、事件覆盖 | — |
| `Var(V_th,E)_surround`（周围阈值离散度） | core vs surround 对比 | **不要只调 core 不定义背景** |

**本轮不把** `V_th,E 平均值↓ / I_E↑ / E_L↑` 与异质性下降**一起**放进主线（会立刻变烂拟合）。第一轮只问：**单独收窄 E 阈值分布，是否能让事件更集中、更稳定、更容易被有限扰动点着。**

**强制 mean-matched control（合同 2）**：每个 `Var(V_th,E)` 收窄实验跑两层 —— (a) raw；(b) mean-rate-matched / mean-drive-matched control。只有 raw 与 control 的差，才是"分布变窄本身"的效应；raw 单独的变化里混着 Jensen 凸性带来的平均工作点移动，不能直接当异质性效应读。

## 6. LIF rate field 版 vs SNN 版（同构对齐）

**同一套底层旋钮，区别只在中观增益怎么"得到"。**

| | LIF rate field | SNN |
|---|---|---|
| 调的参数 | `θ_EE, ℓ_∥, ℓ_⊥, w_EE`；patch 内外 `p(V_th,E)`（尤其方差） | E→E 连接概率 / 权重的空间各向异性；patch 内外 E 神经元 `V_th` 分布 |
| 增益怎么得到 | **算**：Siegert `Φ_LIF` 直接给斜率 / 曲率；异质性**积分掉** `∫Φ_LIF(μ,σ;θ)p(θ)dθ` | **测**：群体 f-I / 扰动响应；异质性**逐神经元铺开** |
| 输出读数 | 有效曲线斜率、曲率、闭环稳定性、有限扰动余量、成核位置、传播方向 | 群体 f-I、扰动响应、**同步突变**、虚拟 SEEG 通道顺序 |
| 分工 | **必要筛查**（粗版：分布变窄 → 曲线 → 闭环放大 + 余量塌缩） | **关键验证**（有限尺寸的真**同步突变** = Rich 的"突然跳进同步"，rate field 平均掉看不见） |

> 异质性↓ 这条**两版都要**：它的"刹不住"在 rate field 是**余量塌缩**，在 SNN 才是真正的**同步爆发**。rate field 不是看不见，是只能抓粗版；SNN 验证真版。

**SNN 同构合同（合同 4，必须锁死否则 SNN 不可解释）**：SNN 里改连接概率 / 权重 / 神经元密度会**同时**改入度、总电流和同步概率 —— 与合同 1 同病，必须显式锁：

```text
① 归一化基准：固定【期望入度 E[k_EE]】 或 固定【总 E→E 电流 ∫ J·p_conn】，择一锁死全程不变
② 改 ρ（各向异性）：重归一化权重，使每个 E 细胞收到的总 E→E 电流不变（只改"来自哪个方向"，不改"收到多少"）
③ 收窄 Var(V_th,E)：保持均值 <V_th,E> 不变（配合合同 2 的 mean-matched）
④ 虚拟 SEEG：必须走真实模板提取 pipeline（与真实数据同一套：通道序、rank template、forward/reverse），不另起炉灶
```

**有效符号由测量定义（次要项，给 deferred 的 GABA 轴备注）**：rate-field `J_EI^eff` 与 SNN 的 `E_GABA + conductance` **非一一等价**（分流可净抑制）。将来实现 GABA 符号轴时，"有效符号"必须由**扰动响应测出**，不是只看 `E_GABA` 是否过某阈。

## 7. 验收：两条待算的问题（不是预设的答案）

### Track 连接（几何，**癫痫中性**，奠基现象）

- 复现 forward/reverse **共轴**模板；**且** isotropic + aligned-shaft **控制必须失败**。
- 承重判据（扩展已锁 0d 测试 `scripts/sef_hfo_step0d_anisotropy_control.py`）：**旋转 `θ_EE` → 模板方向跟着转；旋转电极杆 → 模板方向不变**。各向同性核下无稳定轴（ratio≈1）。

### Track 异质性（**癫痫规范**，第一条方向）

- 收窄 patch 内 `Var(V_th,E)`，**在工作点上算**它对斜率 / 曲率 → 闭环放大 / 安全余量 `S` 的影响 → 是否把可观测（事件集中度、模板稳定性、有限扰动可点燃性）推向文献预测方向。
- **非循环声明**：文献方向（Rich：异质性↓ → 余量压缩 → 事件更集中 / 更易同步）是**待检验假设，不是调参目标**。计算**允许 disconfirm** 它 —— 若在我们的工作点上收窄 `Var(V_th,E)` 并**不**压缩余量（甚至扩张），那是一个**真实结果**（印证"方向取决于工作点"这条修正），不是需要调走的失败。
- **方向必须从计算里出来**，不能写进假设。core vs surround 对比必须存在（不能只调 core）。

### 闭环（两 Track 共用引擎）

- 敏感度计算 `∂(可观测)/∂(参数)` = 强制链每个箭头的符号来源 = 后续辨识性 gating（哪些旋钮能从间期观测分开）的引擎。本轮先在第一轮参数子集上建立这套计算管线。

## 8. 范围 / 红线核对 / 暂不进主线

**本轮范围**：只有 E→E 连接几何 + E 阈值异质性两轴。

**暂不进主线（= 预注册正向预测，留后续轮次）**：抑制连接（surround 自限）、E 平均兴奋性移动、E→I / I 细胞数、GABA 去极化符号翻转、慢状态漂移、细胞丢失 / 稀疏化（方向本身不定，是坏规范旋钮）。

**红线核对**：
1. **§8.1（HFO carrier 不分 E/I）不冲突**：本 spec 的带符号 E/I 耦合与 GABA 符号轴是**群体率 / event-envelope 层**的动力学，**不**对 HFO 80–250 Hz carrier 振荡来源做 E/I 归因。
2. **§8.4（低异质性≠易激性）= 本 spec 正式化**：异质性永远经 `斜率 / 曲率 → 闭环稳定 → 有限扰动余量` 进入解释，**禁止** `Var↓ ⇒ 更易激` 直写。
3. **clinical SOZ 不作拟合标签**：`x_patch` 是模型成核热点，**不**回拟 clinical SOZ；与 SOZ 关系只作 held-out。
4. **ictal recruitment 只作 synthetic feasibility bridge**，不解释 clinical seizure onset。
5. **近临界降级**：统一用韧性 / 余量 `S`（Lepeu），不用临界慢化。

**自由度纪律**：第一轮自由拟合旋钮数必须小（连接 4 个 + 异质性 4 个，其中 `w_EE` 不作方向旋钮）；其余全部 fixed / deferred。

**组织类型 caveat**：Rich = 新皮层 L5；Huberfeld = 内侧海马下托；Lepeu = 海马。异质性↓ / GABA 翻转是**跨组织抽象因子**，不声称单一基质上全成立。
