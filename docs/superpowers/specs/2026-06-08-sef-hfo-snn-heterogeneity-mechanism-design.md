# SEF-HFO 放电网络异质性病理核 —— 机制实现 + 粗网格筛查 Design Spec

> **状态**：draft 2026-06-08。本 spec 是把 `2026-06-06-sef-hfo-pathology-parameter-mapping-design.md` §5.2（E 阈值参差收窄病理核）+ `2026-06-06-sef-hfo-virtual-seeg-observation-layer-design.md` §6（SNN 低驱动竖切读出）**在放电网络（SNN）上落地实现**的第一轮，对应用户 2026-06-08 拍板的"实验 A"。
>
> **关系**：
> - **实现**（不替换）pathology-mapping spec §5.2 的门槛参差收窄轴 + §5.0 调参合同 2（mean-matched control）+ 合同 4（SNN 同构锁）；observation-layer spec §6 增量 2 的"透过虚拟电极 + 真实读出链"。
> - **保留**（不触碰）两份母 spec 已锁的强制链纪律、phantom-rank 纪律、clinical SOZ 不作拟合标签红线、v0.2 核心边界。
> - **承接** rate 模型结论：内在细胞参数包（Θ_cell，含 `Var(V_th,E)`）在率均值场里被封为 null（收窄使有效增益升约 13%，但**不动安全余量**）；存在性移交 SNN，因为率均值场把"有限尺寸的真同步爆发"平均掉了，看不见——这正是 SNN 独有、要测的东西。
>
> **文献锚**（已查证，承自 pathology-mapping spec）：Rich et al. 2022 *Cell Reports*（致痫皮层 L5 锥体细胞生物物理异质性下降 → 更易突然同步）。

## 0. 一句话承诺（朴素话）

到现在为止，放电网络里每个兴奋性神经元都长得一模一样（同一个固定点火门槛，组织内部零差异）。这一轮把"病理异质性"装进去：在一小块**病理核**里，把细胞点火门槛的**参差程度收窄**（细胞之间更齐），看放电网络里会不会出现率均值场看不见的东西——核内细胞是不是更容易一起同步、事件是不是更尖、传得更远/更近、还自不自终止、隔着电极读出来的通道先后顺序变没变。

方法是"实验 A"：同一片组织、同一个戳点，比"均匀组织 vs 带病理核组织"。再做一个**粗网格**——挪戳点、挪病理核——看效应稳不稳、在哪儿最明显，挑几个代表性案例出机制图。

## 1. 实验 A 框架（固定戳点·比传播变化，非循环）

- **问题**：固定戳点、固定连接，比"均匀 vs 病理核（同一片其它都一样）"——病理核改不改传播 / 同步 / 自终止 / 电极读出顺序？直接复检率均值场那条"余量没动"的结论：率均值场看不见的真同步，到底动不动。
- **非循环纪律**：戳点放在传播路径一端，病理核放在路径上、**不压在戳盘上**。网格里"戳在核上"的格子会显式标注——"戳核上、事件从核起"是循环论证，**不当机制证据**；真正有信息的是"戳在核外、波穿过核被改没被改"。
- **方向从仿真读、允许 disconfirm**：文献方向（Rich：参差↓ → 更易同步）是待检假设、不是调参目标。若用锁定的干净参差（1.5→0.5）效应很小甚至看不见，那本身是个结果（与率均值场 null 一致），**不**偷偷放宽截断把效应做大——要更大幅度得改工作点 / 门槛间隙（框架级决定）。

## 2. 门槛场的三种条件（合同 2 的 SNN 版 + 用户加的"不配平"）

工作点全程固定：`L=3.0 mm`、`density=1800 /mm²`（N≈16,200）、`drive=0.6`（安静可激）、E→E 各向异性长轴 `θ_EE=45°`、拉伸 `AR=2.0`（与现有 `two_electrode_readout_snn.png` 一致）。门槛 `V_th=18 mV`、复位 `V_reset=11 mV`、间隙 7 mV（与 rate 模型同间隙，故复用 rate 锁定的参差幅度）。

> **2026-06-08b 局部化修正（实测推翻原设计，不是 bug 是前提被数据否定）**：原设计把宽参差铺满整片（baseline = 整片 `N(18,1.5)`）。实测 kick-OFF：**宽参差整片会自发爆发**（低门槛左尾细胞自点着，299 Hz @65ms + 202 Hz @281ms），而标量整片安静（2 Hz）。自发爆发**摧毁了实验 A"固定戳点 → 干净诱发传播"的前提**（戳出来的"事件"其实是自发的）。修正：**异质性只放病理核里，外围保持标量安静**（已验证：局部宽核 + 标量外围 kick-OFF 安静 2 Hz、kick-ON 在 176ms 干净点着）。"宽参差搅乱安静点"作为旁支发现（B 侧，§6）写进 recap。

| 条件 | 外围（surround） | 核内门槛分布 | 隔离的是 |
|---|---|---|---|
| **基线 baseline** | **标量安静 18** | 核内 `N(18, 1.5)`（宽=健康异质） | 参考系（健康异质核 vs 病理窄核） |
| **配平核 matched** | 标量安静 18 | 核内 `N(18, 0.5)`，均值 18 | **核内参差收窄**（门槛均值配平；**非**发放率配平，仍带残留 Jensen——见下"诚实标注"，不声称纯方差隔离） |
| **不配平核 unmatched / hyperexcitable** | 标量安静 18 | 核内 `N(16, 0.5)`，均值 16（下移 2 mV） | **核内参差收窄 + 局部更易激**（Rich"异质性↓ + 更易激"）。注意：是"**局部更易兴奋**"，**不是细胞死亡**——细胞死亡 = 密度↓ / 连接缺失，本轮明确不做（§8）。命名一律 unmatched / hyperexcitable core，**禁止**写"神经元死了" |

> 三条件**外围 bit-identical（标量 18）**、噪声/kick 同种子，只有**核内门槛分布**不同——配对种子合同（下）在局部化设计下天然满足。对比 = 健康宽核 vs 病理窄核（matched/unmatched）。

- **物理域**：门槛按**截断高斯**（下界 `V_reset=11`，重归一化）取样；`lif`/取样器对 `v_th < V_reset` 报错（loud tripwire）。在 1.5/0.5、均值 16–18 这个量级，11 处截断质量可忽略，但按 pathology-mapping spec §5.2 gap-limit 规矩照做。
- **锁定参差**：`std_wide=1.5`、`std_narrow=0.5`（复用 rate lock，同 7 mV 间隙）。`unmatched` 的均值下移目标 `vth_core_unmatched=16`（约 2 mV，~28% 间隙），eyeball 可调但锁进 spec。
- **驱动一致性**：`compute_nu_theta` 用标量 `p.V_th=18` 定全局外驱 `ν_signal=0.6·ν_θ`；逐神经元门槛只进 spike 比较。配平核（均值仍 18）工作点定义干净；不配平核（均值 16）是**有意**让工作点局部移动，是该条件的一部分。
- **诚实标注（统一口径，全文以此为准）**：配平核**只是"门槛均值配平"，不是"发放率配平"**。非线性曲线下，同样平均门槛、不同方差仍带 **Jensen 效应**（平均发放率会动）。所以配平核**不能**声称"纯隔离参差本身"——它隔离的是"参差收窄 + 残留 Jensen"。rate 模型另分过"发放率配平"层来去掉 Jensen，本轮 SNN **不做**率配平，recap 必须明说：SNN 这一轮量的是"参差收窄 + Jensen + 有限尺寸同步"的合并后果。
- **配对随机种子（硬合同）**：baseline 与病理核两次跑必须**同连接、同 OU 噪声流、同 kick、核外门槛场逐神经元 bit-identical**——只替换**核内**神经元的门槛分布。取样器构造：先按种子抽出整片宽参差场 `W`；baseline 用 `W`；病理核条件用 `W`、但把**核内**神经元换成窄分布抽样（配平=均值18 / 不配平=均值16），核外保持 `W` 不变。否则网格里的差异可能只是随机门槛抽样噪声、不是病理核效应。每个网格格子的效应都对**同种子的 baseline** 取差值（within-seed Δ）。

## 3. 粗网格（两条扫法，电极固定，~12–18 跑）

- **扫法 1（固定核、动 kick）**：病理核钉在传播路径中段，戳点取 ~5 处代表位置（核上 / 核旁 / 对端 / 偏轴 / 近电极）。
- **扫法 2（固定 kick、动核）**：戳点钉在轴一端，病理核位置取 ~4 处（近种子 / 中段 / 远端 / 偏轴）× {配平, 不配平}。
- 合起来 = **(病理核配置 × kick 点) 粗网格**；电极杆（∥/⊥ 那对）位置**全程固定**。
- 每个 kick-ON 配一个 kick-OFF 自发对照（引擎已有）。
- **先计时一次 L=3mm 跑**，按 ~12–18 预算定最终点数；长就后台跑。

## 4. 量什么 + 怎么挑代表

- **网格每格效应 = 源空间（上帝视角）度量**，与电极无关、所有格可比：
  - 峰值发放率 `window_peak_rate`、峰值"同时活跃"细胞比例 `peak_active_fraction`、核外招募 `spk_outside`、是否自终止 `returned`（引擎 `compute_metrics` 已有）；
  - 源空间传播：E 神经元首发时刻梯度的主轴角 + 核内同步度（核内细胞在小窗内一起发放的比例）。
  - 每格对**基线**取差值（Δpeak_active_frac、Δ轴角、Δ核外招募…）。
- **代表案例选择规则（预注册——先全网格、后挑，不挑好看的图）**：**必须先出全网格总览图**，再按这条写死的规则挑 3–4 个代表：① 最大效应格 ② 近零效应格 ③ 戳核外·波穿核的干净格 ④ 配平 vs 不配平对照。规则锁进 runner / 本 spec，**不**事后按"图好不好看"换。每个代表出完整机制图（含电极读出）。
- **电极覆盖 caveat**：电极固定 → 部分格子事件离电极远、读出（b/c 面板）信息少。故**网格度量用源空间**（所有格可比），**电极读出只在有覆盖的代表案例上展示**（与 observation-layer spec：oracle 机制检查 + 观测读出分层一致）。

## 6. 白话 recap 结构（强制链，两条轴都覆盖）

- **连接方向轴**（已实现、癫痫中性）：决定往哪传；图里就是长轴箭头。
- **门槛参差轴**（新加、Rich 病理）：底层旋钮 = 核内参差↓（不配平再叠均值↓）→ 该影响的中间层度量（率均值场预言：曲线斜率↑约13% / 安全余量不动；跨细胞离散度↓是构造出来的）→ 放电版**实测**：核内是否更同步、事件更尖否、传得远近、自不自终止、电极读出顺序变没变。
- recap 必须明说：率均值场已说这条对余量是 null；放电版来看"平均抹掉的真同步动不动"，方向从仿真读、允许 disconfirm；网格"戳核上"格子标注、不当机制证据。

## 8. 本轮范围 / 不做

- **做（主线 = 修正实验 A）**：局部化门槛参差病理核（外围标量安静 + 核内 配平/不配平）；粗网格（kick × 核）~12–18 跑（事件诱发，窗 [150,350]）；源空间网格度量；3–4 代表机制图；基线对比图；白话 recap。
- **做（旁支 = B 侧发现）**：把"宽参差整片 → 自发有限尺寸爆发"作为 documented finding（kick-OFF：宽参差 299 Hz 自发 vs 标量 2 Hz 安静）写进 recap + 一张诊断图。放电版看见、率均值场看不见的东西。
- **做（稳健性，advisor 加固）**：代表对（mid 核 配平 vs 不配平）跑 3 个随机种子报 mean±std；其它格子单种子（粗筛、诚实标注）。
- **不做（留下一步）**：厘米级大规模参数标定；细胞丢失（密度↓）轴；率配平层；全网格多种子；正式 cohort 级"均匀 vs 异质"科学裁决（成核集中度 / 模板数量 vs ~2 的统计裁决）；rate-field 异质核 parity。
- **红线**：gap-limit 不偷偷放宽；clinical SOZ 不作拟合；近临界统一用韧性 / 余量措辞。
