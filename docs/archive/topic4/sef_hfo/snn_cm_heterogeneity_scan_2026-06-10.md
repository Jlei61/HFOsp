## 0. 朴素话三段

**测了什么**：在能插真实 4mm 电极、且"戳→沿连接传→读方向"四对照已经全过的厘米尺寸放电网络上，挖一块
"病理核"——把核内神经元的点火门槛调低（更易兴奋）并加上离散度（参差）——看：(a) 这块核要不要外界"戳"才点着，
还是自己就自发点着；(b) 点着以后传出去的波，虚拟电极能不能读出方向。

**怎么测的**：同一套读出 + 同一工作点，只在核内改门槛。核的"均值往下移多少 mV"和"参差宽窄"两个旋钮各扫几档。
每格跑两遍：踢一下（on）+ 不踢（off）。**不踢也点着 = 自发病灶；不踢安静、踢了才点 = 诱发**。再让虚拟 SEEG 读方向。

**揭示了什么**：①**裸片（不挖核）不踢就安静**（活动占比 0.0004），踢一下才出事件（0.071）——所以"核自发点着"
是核自己造成的，不是整片过度兴奋的假象。②**核只要把门槛均值往下移 ≥1mV，就自发点着**（变成自发病灶），
核大核小都一样（587 个神经元的大核、64 个的小核都自发）。③**门槛均值不动、只加参差（宽/窄都行）的核，不踢就安静、
踢了才点**——这是干净的诱发档。④**不管自发还是诱发，传出去的波虚拟 SEEG 都读出 ~45°的连接方向**（两种读法都
≥10/18 触点、误差 ≤10.9°）。⑤这跟 2mm 小尺度刻画的"均值闸门"机制**一模一样**（只加参差=诱发；均值下移=自发），
只是厘米尺度的自发门槛更低（<1mV vs 小尺度的 1.1–1.7mV）——大网络有限尺寸噪声更小，更容易自发，符合 scale law。

## 1. 工作点 + 方法（复用，不新建）

- **片 + 读出（验收过，不动）**: L=20mm, density=100（N≈40k, 13GB 内存可行）, g=3.6/drive=0.6/θ_EE=45°/AR=2,
  r_kick=0.6, 虚拟 SEEG = 3 杆×nc6×4mm = 18 触点。四对照（C1/C2/C3/C4）已在此闭合（见 [[snn_cm_traveling_wave]]）。
- **核半径**: 实测 d=100 的 E→E 有效触达 ~1.46mm（mean partner-distance），故核半径取 1.5mm（=1×触达）为主，
  另测 0.5mm（≈L=3 核大小，亚触达）作对照。**核半径是算出来的不是盲扫的**（advisor）。
- **唯一经验轴**: core_mean × core_std。core_mean=18（与周围同=纯参差）→ 17/16/15/14/13（下移 1–5mV）；
  std wide=1.5 / narrow=0.5。核放在踢点起点（核即点火源）。

## 3. 服务 Step-3 的参数范围（结论）

**两个都读得出方向，按"要不要踢"分两档**：

- **诱发-干净档（Step-3 读出即用）**: `core_mean=18`（均值与周围齐平，**只有参差**异质性），spread wide/narrow 皆可，
  核半径不限。不踢安静、踢了才点，传播读出 45°（两读 10/18，误差 10.9°），evoked_clean。**读出层已验收，可直接喂 Step-3**。
  科学口径提醒：这是"参差是否调制**诱发**传播"那条腿——L=3 上该腿结论是"跨位置无一致方向"；这里读得出方向（方向由连接定，
  非核定），但"参差改不改变传播"仍是那条不确定的腿，别当新结论。
- **自发-病灶档（病理更真实，但 NOT-YET）**: `core_mean ≤ 17`（均值下移 ≥1mV），核大小不限。自发点着（=自发病灶，
  更像真实致痫灶），传播也读出 45°。**但目前 seed=1 单次实现**——自发点火的概率/潜伏/位置等定量主张需 multi-seed +
  finite-size 才够格（[[snn_heterogeneity_result]] 的 B-side 纪律：单实现 = 风险警报级）。**自发档的"会不会点着"
  本身极稳**（ign_off 0.07 vs 裸片 0.0004 = 4 量级，非边界判定），但"点着的统计"要多种子。

**均值闸门边界**: 自发临界在 core_mean 18（诱发）与 17（自发）之间，即厘米尺度自发门槛 = **<1mV 下移**；
比 L=3/d1800 的 ~1.1–1.7mV **更低**——大 N 有限尺寸噪声更小→更易自发，符合 scale law。

**机制可迁移性**: cm SNN **复现了** L=3 的均值闸门机制（纯参差→诱发；均值下移→自发），双模态读出对异质性事件成立。

## 4. 防坑 + caveat

- **confound 已排除**: 裸片 off=0.0004（安静），自发是核驱动非整片近临界。
- **core size 不开诱发窗**: 小核（r=0.5, 64 神经元，≈L=3 大小）在均值下移时照样自发——开诱发窗的是"均值不下移"
  （纯参差），不是"核够小"。
- **single-seed**: 本扫只 seed=1。诱发-vs-自发的**档位分类**极稳（4 量级隔离，非边界），但自发档的定量统计需 multi-seed。
- **d=100 长程**: 触达 1.46mm（vs L=3 的 0.86mm），连接比真实皮层长程——这是为塞进 cm 内存的工程取舍，"对 L=3 0.38mm
  触达的保真"本就是对一个 screen 的保真、不是对生物的保真（不拿它当密度选择依据）。
- 阈值/估计子/reframe/framework 未动；工作点 g=3.6/drive=0.6 跨尺度不变。
