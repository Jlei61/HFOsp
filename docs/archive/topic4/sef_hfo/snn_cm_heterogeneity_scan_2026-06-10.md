## 0. 朴素话三段

**测了什么**：在能插真实 4mm 电极、且"戳→沿连接传→读方向"四对照已经全过的厘米尺寸放电网络上，挖一块
"病理核"——把核内神经元的点火门槛调低（更易兴奋）并加上离散度（参差）——看：(a) 这块核要不要外界"戳"才点着，
还是自己就自发点着；(b) 点着以后传出去的波，虚拟电极能不能读出方向。

**怎么测的**：同一套读出 + 同一工作点，只在核内改门槛。核的"均值往下移多少 mV"和"参差宽窄"两个旋钮各扫几档。
每格跑两遍：踢一下（on）+ 不踢（off）。**不踢也点着 = 自发病灶；不踢安静、踢了才点 = 诱发**。再让虚拟 SEEG 读方向。

**揭示了什么**：①**裸片（不挖核）不踢就安静**（活动占比 0.0004），踢一下才出事件（0.071）——所以"核自发点着"
是核自己造成的，不是整片过度兴奋的假象。②**自发 vs 诱发的分界，由"参差宽窄"主导，不只看均值**：核内门槛
**宽参差**的，只要均值往下挪 ~0.1mV（core_mean 17.9）就自发点着；**窄参差**的要挪到 ~0.9mV（core_mean 17.1）
才自发——宽窄两条分界**差 ~0.8mV，且 3 个种子完全一致（宽 3/3 自发、窄 0/3）**。这正是 2mm 小尺度那条
**承重的"宽参差的低门槛尾巴更容易点火"**对照（小尺度 11/12 vs 3/12），**在厘米尺度复现了**。③**每条分界本身很陡**
（同一参差下，均值挪 ≤0.2mV 就从全静默跳到全自发），**不是小尺度那种 0.6mV 宽的渐变**——大网络有限尺寸噪声更小，
分界更锐，符合 scale law（机制同向、形状变锐）。④**门槛均值不下移、只加参差的核（窄参差可到 17.2、宽参差仅 18）
不踢就安静、踢了才点**=干净诱发档。⑤**不管自发还是诱发，传出去的波虚拟 SEEG 都读出 ~45°连接方向**（两读 ≥10/18
触点、误差 ≤10.9°）——读出层对异质性事件成立。

## 3. 服务 Step-3 的参数范围（结论，细扫后定稿）

**分界由"参差宽窄"主导（非单看均值），陡、且 3 种子稳。两档都读出 45°方向。**

- **诱发-干净档（Step-3 读出即用，kick 需要）**: 核内门槛不下移到自发线以下即可——**窄参差 core_mean ∈ [17.2, 18]**
  （下移 0–0.8mV 仍不自发）、**宽参差仅 core_mean = 18**（再降即自发）。此档不踢安静、踢了才点，传播读 45°
  （两读 10/18，误差 10.9°），evoked_clean。**读出层已验收，可直接喂 Step-3**。科学口径提醒：这是"参差是否调制
  **诱发**传播"那条腿——L=3 上该腿是"跨位置无一致方向"；这里读得出方向（方向由连接定非核定），"参差改不改变诱发传播"
  仍是那条不确定的腿，别当新结论。
- **自发-病灶档（病理更真实）**: **宽参差 core_mean ≤ 17.8、窄参差 core_mean ≤ 17.0** → 自发点着（=自发致痫灶），
  传播也读 45°，核大小不限（587/64 神经元都自发）。**"会不会点着"极稳**（ign_off 0.07 vs 裸片 0.0004=4 量级，
  且 3 种子一致），但**自发点火的概率/潜伏/位置等定量主张仍需更系统的 multi-seed + finite-size**（[[snn_heterogeneity_result]]
  B-side 纪律）——本扫每格 3 种子够定"分界位置"，不够定"自发统计分布"。

**自发分界（细扫定稿）**: **宽参差 ≈ 17.9（下移 ~0.1mV）、窄参差 ≈ 17.1（下移 ~0.9mV）**，宽窄差 **~0.8mV**。
比 L=3/d1800 的 ~1.1–1.7mV **更低**（大 N 有限尺寸噪声更小→更易自发，scale law）。

**机制可迁移性（精确口径）**: cm SNN **复现了 L=3 的承重对照——"宽参差低门槛尾更易点火"**（宽分界比窄高 ~0.8mV，
3 种子稳）。但 **cm 分界是陡的（≤0.2mV）、不是 L=3 那种 0.6mV 渐变**——所以复现的是 wide-vs-narrow 的**方向/排序**，
不是 graded 的**形状**；形状变锐符合 scale law（大 N 噪声小）。**不写"完全复现 L=3 均值闸门"**——复现排序、形状变锐。

## 4. 防坑 + caveat

- **confound 已排除**: 裸片 off=0.0004（安静），自发是核驱动非整片近临界。
- **core size 不开诱发窗**: 小核（r=0.5, 64 神经元，≈L=3 大小）在均值下移时照样自发——开诱发窗的是"均值不下移"
  （纯参差），不是"核够小"。
- **种子**: 粗扫（§2）seed=1；**自发分界细扫（§2.5）用了 3 种子，分界位置 + 宽窄分离 3/3 一致**=稳。
  但自发档的**定量统计**（点火概率分布/潜伏/位置）仍需更系统 multi-seed + finite-size（3 种子够定分界、不够定分布）。
- **d=100 长程**: 触达 1.46mm（vs L=3 的 0.86mm），连接比真实皮层长程——这是为塞进 cm 内存的工程取舍，"对 L=3 0.38mm
  触达的保真"本就是对一个 screen 的保真、不是对生物的保真（不拿它当密度选择依据）。
- 阈值/估计子/reframe/framework 未动；工作点 g=3.6/drive=0.6 跨尺度不变。
