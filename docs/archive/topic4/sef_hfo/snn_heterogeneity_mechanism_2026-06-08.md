## 摘要（白话三段）

**测了什么**：我们把"一小块组织变得更整齐"（核内细胞的点火门槛参差变小，对应文献 Rich 说的致痫组织异质性下降）装进一个会逐个神经元放电的网络，看它对一个"戳一下→定向铺开→自己停下"的传播事件有什么影响：核里细胞同不同步、传播方向变不变、事件提前还是推后、整个网络会不会失控。

**怎么测的**：同一片组织、同一个戳点、同一串噪声，只把那一小块核的门槛分布换掉，跟"健康宽参差核"逐一配对比。核做两种病理化：① 只把参差收窄、平均门槛不动（"配平"）；② 参差收窄、同时把平均门槛也压低（"不配平"=整块更易兴奋）。外围组织保持均匀安静（标量门槛），这样不戳时网络是安静的、戳了才点着。最关心的那一对（核配平 vs 不配平）跑了 3 个随机种子，确认不是单次噪声。

**揭示了什么**：
- **光把参差收窄、平均不动——几乎什么都没变**：核里同步度 −0.01±0.04（3 种子，等于零），传播方向不变（差 0–19°），事件不提前不推后，网络照样自己停。也就是"健康组织失去异质性"这一条**单独拿出来、在这个间期工作点上，看起来不直接起作用**——这跟"抹平平均"的数学早先算的"收窄不动安全余量"是一致的，而且讲得更细。
- **真正起作用的是"平均更易兴奋"那一半**：把核的平均门槛压低后，核会**自己提前点着**（戳之前 ~80ms 就烧起来，成了一个局部"起火点"），核里同步度大变（中心位置稳定 +0.51±0.04），但好坏取决于核摆在哪——摆在远端反而把主传播波打散、同步**下降**。
- **一句话**：在我们这个间期工作点上，"病理 = 失去异质性"本身（参差变窄）**不**直接让组织更易同步；让它更易同步、更易自发点着的是"平均更易兴奋"。这两件事要分开说，不能揉成一句"异质性下降导致癫痫"。
- **旁支发现**：如果不把异质性局限在核里、而是把宽参差铺满整片组织，整片就**自发一阵阵爆发**了（不戳也有 328 Hz，而均匀组织只有 2 Hz 安静）。这是"逐个放电"版看得见、而"抹平平均"的数学看不见的东西（它把"很多细胞突然一起同步"平均掉了）。

**没看清 / 没做的**：除了中心那一对（3 种子），其它格子都是单次实现（粗筛），数值有噪声、只看方向不下定量裁决；都是小尺度（3mm 片、电极按比例缩小、不是真实厘米级 SEEG）；不配平核的效应跟核位置强相关，没做系统的位置扫描；这一轮只做机制图 + recap，没做 cohort 级统计裁决。

## 1. 设计纠偏（先错后改，记录在案）

第一版按 spec 原设计把宽参差（std=1.5，率模型锁定的"干净"值）**铺满整片**。目视 baseline-vs-core 群体轨迹时发现事件出现在 ~300ms（戳在 150ms）、还有戳前 ~70ms 的峰。kick-OFF 对照确认：**宽参差整片不戳也自发爆发**（299→328 Hz @65ms + 202 Hz @281ms），均匀标量整片安静（2 Hz）。结论：宽参差的低门槛左尾（门槛低到 ~13.5 mV、接近静息驱动）自点着 → 整片从"安静可激"变"自发活跃" → 原网格里的"事件"其实是自发的、"读传播方向"失去意义，且度量窗 [150,300] 还把延后的事件切掉（窗口偏置）。

修正（user 拍板"两个都要"）：主线把**异质性只放病理核、外围保持标量安静**（已验证：局部宽核 + 标量外围 kick-OFF 安静 2 Hz、kick-ON 在 176ms 干净点着），恢复"固定戳点 → 诱发传播"实验 A；旁支把"宽参差→自发爆发"作为 documented finding。度量窗放宽到 [150,350]（max-based 不稀释）捕住事件；图读出窗改成事件锁定。

> 教训：单次实现 + 固定窗 会把"事件时移"读成"同步变化"（第一版误判 matched 核降低同步，其实是配平核把事件推后 +7ms、被 300ms 窗切掉的伪信号；干净窗 + 3 种子后 matched = 0）。eyeball 基线对比图是抓到这个的关键。

## 2. 主结果（局部化实验 A，L=3mm，drive 0.6，θ_EE=45°/AR=2）

**三种核**（外围标量 18 安静，核 r=0.5mm ~1167 E 细胞）：baseline 核 = N(18, 1.5) 宽健康；matched = N(18, 0.5) 窄、均值不动；unmatched = N(16, 0.5) 窄 + 均值下移。

**自终止**：12 格 base + core **全部 returned=True**（病理核不致失控）——与率均值场"余量不塌"一致。

**核内同步（d_core_paf = core − baseline，3 种子 mid 核）**：
| 条件 | d_core_paf (mean±std, 3 seeds) | 解读 |
|---|---|---|
| matched（仅收窄参差） | **−0.013 ± 0.049** | ≈ 0，参差本身无效应 |
| unmatched（收窄 + 均值下移） | **+0.515 ± 0.042** | 大且稳定，效应来自"更易激" |

**事件时移（d_event_t，mid 3 种子）**：matched +0.2±1.5 ms（≈0）；unmatched +80±73 ms（更易激核自发早点着，主峰被推后，方差大）。

**传播方向（d_axis_deg，off-patch grid）**：matched 核 0–19°（基本不动）；unmatched 18–48°（扰动更大）。

**核位置依赖（off-patch grid，单种子）**：matched 各位置 d_core_paf ∈ [−0.08, +0.17]（都贴近 0）；unmatched 强依赖位置 mid +0.54 / offaxis +0.28 / far **−0.57**（远端核把主波打散）。

**机制读图（baseline_compare）**：matched 核群体轨迹与基线几乎重合（单干净事件 ~185ms）；unmatched 核多出 ~80ms 早峰 = 更易激核戳前自点着 = 局部成核热点。

## 3. B 侧旁支：宽参差 → 自发有限尺寸爆发

kick-OFF（无刺激）：均匀标量 2.3 Hz 安静；宽参差铺满整片 327.9 Hz @65.5ms 自发爆发（`spontaneous_probe.json`）。放电版独有的有限尺寸失稳——率均值场把这种"很多细胞突然一起同步"平均掉，看不见（率模型说 1.5→0.5 只动斜率 +13%、余量不动；放电版说 1.5 本身就让组织自发活跃）。这是"存在性移交 SNN"的直接体现，也是主实验局部化（外围安静）的动机。

## 4. 与强制链 / 红线的对账

- **方向从计算出、允许 disconfirm**：文献 Rich"异质性↓→更易同步"在这个工作点上、单独看参差时被**证否**（matched=null）——这是真实结果，不是失败。同步/易激的增益归到"平均更易兴奋"（mean shift），与 spec §7"非循环声明"一致。
- **配平核 ≠ 纯方差隔离**：matched 是门槛均值配平、非发放率配平，仍带残留 Jensen；本轮量的是合并后果（恰好 ≈ 0）。
- **不配平核 = 局部更易激，不是细胞死亡**：unmatched 是 mean down（hyperexcitable），细胞丢失（密度↓/连接缺失）本轮不做。
- **gap-limit 未放宽**：std 仍锁 1.5/0.5；均值下移 2mV（间隙 7mV 的 ~28%）。

## 5. 诚实 caveat

- 单种子粗筛（除 mid 对 3 种子）；数值有噪声，只下方向判断不下定量裁决。
- 小尺度（L=3mm、电极按比例缩小、firing-density 读出非 LFP、非真实 SEEG 间距）。
- unmatched 效应强位置依赖，未系统扫位置。
- 机制图局部离散度上色：unmatched 核边界有平均跳变伪环（非核内真实参差，看核内部）。
- 引擎改动（kick_probe.py V_th_per_neuron）gitignored，已 patch + checksum + runner 断言追踪（`engine_versions.json` / `engine_patch/`）。
