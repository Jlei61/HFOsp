## 摘要（白话三段）

**测了什么**：我们把"一小块组织变得更整齐"（核内细胞的点火门槛**离散度**变小——这是把文献 Rich 的宽义异质性 operationalize 成"局部点火门槛离散度"这一个**窄 projection**，不等于全部异质性）装进一个会逐个神经元放电的网络，看它对一个"戳一下→定向铺开→自己停下"的传播事件有什么影响：核里同不同步、方向变不变、事件提前推后、会不会失控、会不会自发点着。

**怎么测的**：同一片组织、同一个戳点、同一串噪声，只把那一小块核的门槛分布换掉，跟"健康宽参差核"逐一配对比。核做成一个 **2×2**（平均门槛 {18, 16} × 参差 {宽, 窄}）：① 只收窄参差、平均不动（"配平"=纯方差轴）；② 只压低平均、参差仍宽（"只均值"=纯均值轴）；③ 两个一起（"不配平"=合并）。**加"只均值"这一格，才能干净分开"参差本身"和"平均更易兴奋"两件事**（review 加固）。外围保持均匀安静（标量门槛），不戳时安静、戳了才点着。mid 核三种病理化各跑 3 个随机种子。自限用干净的"戳前静息窗 + 衰减比 + 持续时长"判（旧的 [50,150] 参考窗会被戳前自点火污染，已修）。每个结果带 provenance（git hash + 引擎 checksum + 配置 + 种子 + 度量版本）。

**揭示了什么**：
- **点不点着，主要由"平均"门控；"参差"宽窄不是主开关，但会调节点火"可靠性"**（2026-06-09 多种子+位置 sweep 修正了早先"参差零影响"的过硬措辞，见 §7）：在这个工作点、这组参数幅度（mean 18→16、std 1.5→0.5、核 0.5mm）下——只收窄参差、平均不动（配平核）**从不自发点着**（84/84 位置-种子格 + grid 8/8 + 3 种子全 0）；把平均压低后，核**大多自发点着**——其中宽参差核（只均值）84/84 **全点着**，窄参差核（不配平）77/84 点着、偶尔点不着。→ **平均门槛下移是自点火的主要门；宽参差里那几个"极低门槛尾巴细胞"帮忙引燃、让点火更保险，把参差收窄会略微降低点火概率**（与下面 B 侧"尾驱动成核"同向）。**不能写成"参差对点火零作用"**。（分层：这是"点不点着"这个二元/概率结果；"点着之后同步量级"是否还受参差调制——本轮位置噪声盖过、**没看清**，见 §7。）
- **光收窄参差、平均不动——诱发事件没有一致方向的改变**：核里诱发同步度跨 7 个位置 × 12 种子**没有一致方向**（位置依赖正负摆动 −0.18 → +0.22，整体 pooled mean −0.023±0.026 ≈ 0，见 §7），传播方向不变（0–19°），事件不提前推后，照样自己停。**说法收紧**：不是"稳健的零"，而是"**跨位置无稳定同向效应**"。"健康组织失去异质性"这一条**单独看、在这个间期工作点、这个尺度、这个基质、std 1.5→0.5 局部核条件下没有复现一个稳定方向的效应**（**不是**证否一个为不同基质/尺度提出的假设）。跟率均值场"收窄不动余量"一致并更细。
- **压低平均 → 局部自点火灶（不是"同一事件更同步"）**：只均值核 / 不配平核把核变成戳前 ~55–80ms 就自己烧起来的局部起火点。图里 d_core_paf 的大数（如不配平 +0.49）**只是这个"自点火状态切换"的粗指标，不能读成"同一个被戳出来的事件变得更同步"**——它们的主爆发在戳之前、在度量窗外，已被 hatched 标出、与配平的诚实诱发数分开。这个数在种子间还很不稳（只均值 +0.22±0.25 单点；sweep 进一步确认它被**核位置**主导、跨位置 −0.28→+0.49、mean_only 与 unmatched 重叠分不开，见 §7），可靠的说法是"它点着了"，不是那个数。
- **都自己停**：16 个格子（含所有自点火的）在干净自限指标下**全部 returned**（核不致失控），与率均值场"余量不塌"一致。
- **一句话**：在这个间期工作点上，把核**变成自发起火灶的主门是"平均更易兴奋"**（参差变窄**单独**做不到，反而因为削掉低门槛尾巴而略微更难点着 77/84 vs 84/84）；而"参差变窄"**单独**也没在诱发传播上造出稳定同向效应。所以"失去异质性"与"平均更易兴奋"两件事要分开——平均管点不点着的主门、参差只调点火可靠性（不是零作用）。
- **旁支（独立机制，不并进"失去异质性"链）**：把宽参差**铺满整片**则整片自发爆发（328 Hz vs 均匀 2 Hz）。这是一条**独立的"尾驱动成核 tail-driven nucleation"机制**——**宽**门槛分布的低门槛稀有细胞能造有限尺寸成核点（注意方向：这是"变**宽**也能造起火点"，跟"变窄危险"不是同一命题，不能混进一条机制链）。放电版看得见、率均值场看不见。

**没看清 / 没做的**：除 mid 核 3 种子外都是单次实现（粗筛），"−0.01±0.05 / +0.5"这种数只够机制筛查、不够论文统计（需 ≥10–20 种子 + 位置/幅度 sweep，见 next-step 路线图）；**纯方差轴还需更干净的控制**：配平是门槛均值配平、非发放率/F-I 斜率配平，仍带 Jensen 残留——加 mean-only 轴让我们能把"平均"和"参差"两条轴分开问，结论是**点火主门是平均下移；参差宽窄调节点火可靠性（不是零）；点着后量级是否有 mean×variance 交互仍没看清**（不写"纯 mean"——那会抹掉 84/84 vs 77/84 的参差作用），完整"纯方差轴"还需率配平 / F-I 配平（roadmap 优先级 4）；都是小尺度（3mm、电极按比例缩小、读出是 firing-density **不是 LFP**，要对应 SEEG 需 forward LFP/HFO 层）；自点火核的效应位置依赖，没系统扫。

## 1. 设计纠偏（先错后改，记录在案）

第一版按 spec 原设计把宽参差（std=1.5，率模型锁定的"干净"值）**铺满整片**。目视 baseline-vs-core 群体轨迹时发现事件出现在 ~300ms（戳在 150ms）、还有戳前 ~70ms 的峰。kick-OFF 对照确认：**宽参差整片不戳也自发爆发**（299→328 Hz @65ms + 202 Hz @281ms），均匀标量整片安静（2 Hz）。结论：宽参差的低门槛左尾（门槛低到 ~13.5 mV、接近静息驱动）自点着 → 整片从"安静可激"变"自发活跃" → 原网格里的"事件"其实是自发的、"读传播方向"失去意义，且度量窗 [150,300] 还把延后的事件切掉（窗口偏置）。

修正（user 拍板"两个都要"）：主线把**异质性只放病理核、外围保持标量安静**（已验证：局部宽核 + 标量外围 kick-OFF 安静 2 Hz、kick-ON 在 176ms 干净点着），恢复"固定戳点 → 诱发传播"实验 A；旁支把"宽参差→自发爆发"作为 documented finding。度量窗放宽到 [150,350]（max-based 不稀释）捕住事件；图读出窗改成事件锁定。

> 教训：单次实现 + 固定窗 会把"事件时移"读成"同步变化"（第一版误判 matched 核降低同步，其实是配平核把事件推后 +7ms、被 300ms 窗切掉的伪信号；干净窗 + 3 种子后 matched = 0）。eyeball 基线对比图是抓到这个的关键。

## 2. 主结果（局部化实验 A，L=3mm，drive 0.6，θ_EE=45°/AR=2）

**四种核 = 2×2**（外围标量 18 安静，核 r=0.5mm ~1167 E 细胞；平均门槛 {18,16} × 参差 {宽,窄}）：

| | 宽参差 std=1.5 | 窄参差 std=0.5 |
|---|---|---|
| **均值 18** | baseline（健康） | matched（**纯方差轴**） |
| **均值 16** | mean_only（**纯均值轴**） | unmatched（合并） |

**自发点火（戳前 [20,150]ms 是否越 10Hz；2026-06-08c 加的干净指标）= 本实验最干净的分离**（grid 3 种子 + 2026-06-09 sweep 12 种子 × 7 位置一并确认，见 §7）：**matched 从不点着**（grid 8/8 + 3 种子 + sweep 84/84 全 0）；**压低平均后大多点着**——mean_only（宽）sweep **84/84 全点着**，unmatched（窄）sweep **77/84 点着**（grid 3 种子里恰好都点着）。→ **平均门槛下移是自点火的主要门（mean-gated）；参差宽窄不是主开关，但调节点火可靠性——宽参差的低门槛尾巴让点火更保险，收窄略降低点火概率**（回答 review 2B/3B 的"点火源"，并修正早先"参差零影响"的过硬措辞）。

> **措辞收紧（advisor + 2026-06-09 review 加固）—— 分两层，别越界**：(a) **点不点着** = 平均为主门、参差调节可靠性（宽 84/84 vs 窄 77/84，方向 = 宽尾巴帮点火）——**不写"参差零作用"**；(b) **点着之后核内同步度的"量级"是否被参差调制 = 没看清**：sweep 里点着后的"烧多旺"指标被**核位置**主导（−0.28 → +0.49 随位置摆动），mean_only 与 unmatched 两条线基本重叠、**分辨不开**。所以**不能**下"参差完全没多加 / 纯 mean、无 mean×variance 交互"的结论——那是把"测不出交互"误当成"没有交互"。能下的只有"点火是平均为主门控的（参差调可靠性）；点着后量级是否受参差影响，被位置噪声盖过、本轮没看清"。

**自终止（干净指标 rest[20,50]+衰减比 decay-ratio，替换旧 [50,150] 弱窗——review 1A）**：16 格 base+core **全部 returned**（含所有自点火核）——核不致失控，与率均值场"余量不塌"一致。旧 `returned` 用 [50,150] 作参考，会被戳前 ~80ms 自点火污染（参考被抬高）；现用戳前静息窗 + 衰减比 + 持续时长，对自点火核也成立。

**核内 peak-active-fraction（d_core_paf = core − baseline，3 种子 mid 核）**：

| 条件 | d_core_paf (mean±std) | 自点火? | 该数含义 |
|---|---|---|---|
| matched (18,窄) | **−0.013 ± 0.049** | 否（诱发） | **诱发事件同步度** ≈0，纯方差无效应 |
| mean_only (16,宽) | +0.225 ± 0.246 | 是 | **自点火状态粗指标**（非诱发同步），种子间不稳 |
| unmatched (16,窄) | +0.494 ± 0.038 | 是 | 同上自点火指标；与 mean_only 同为"点着" |

> 关键（review 1B）：只有 matched 的 d_core_paf 是"诱发事件同步度"（它不自点火）；mean_only/unmatched 自点火，其 d_core_paf 是状态切换的粗指标、与 matched 的诱发数**不可直接比**（图里已 hatched 标出）。可靠说法是"它点着了"，不是那个数。

**传播方向（d_axis_deg，off-patch）**：matched 核 0–19°（基本不动）；自点火核扰动更大且位置依赖。**降级纪律（2026-06-09 review，与 d_core_paf 同处理）**：一旦核在戳之前已自点着，后面再算的 post-kick 传播轴、同步数、事件时移**都不是同一个"被戳出来的事件"**了——自点火格子的 post-kick 传播轴**只能作图示，不进机制结论**；只有 evoked_clean（戳前不点着）的格子，其传播轴才是诱发传播的有效读数（runner 已打 `evoked_clean` 标）。

**位置依赖（off-patch grid，单种子）**：matched 各位置 d_core_paf ∈ [−0.08, +0.16]（都贴近 0，稳）；自点火核 d_core_paf 强依赖位置、正负翻转（mid/nearseed/offaxis 正、far 负）——这是**自点火状态指标的位置噪声**，不是诱发同步的位置规律。

**机制读图**：`baseline_compare`（源空间群体率）—— matched 与基线几乎重合（单干净诱发事件 ~185ms，戳后）；mean_only / unmatched 在戳之前 ~55–80ms 就有早峰 = 自点火灶（戳线 @150ms 之前，一眼可辨）。`propagation_*`（onset 上色 + 电极按到达时刻上色 + 峰轨迹斜率）展示传播顺序；`heterogeneity_*`（plasma 上色，与 onset 区分）单独展示核异质性。grid_overview 用 hatch 标自点火格（mean-down 全 hatch，variance 全不 hatch）。

## 3. B 侧旁支：尾驱动成核 tail-driven nucleation（独立机制，**不**并进"失去异质性"链）

kick-OFF（无刺激）：均匀标量 2.3 Hz 安静；宽参差铺满整片 327.9 Hz @65.5ms 自发爆发（`spontaneous_probe.json`）。机制是：**宽**门槛分布的低门槛稀有细胞（左尾，门槛 ~13.5mV、接近静息驱动）自点着，造有限尺寸成核点。放电版独有的有限尺寸失稳——率均值场把"很多细胞突然一起同步"平均掉，看不见（率模型说 1.5→0.5 只动斜率 +13%、余量不动；放电版说宽尾本身就让组织自发活跃）。

> **方向纪律（review D）**：这条是"**变宽**也能造起火点（尾驱动成核）"，跟主线"变窄危险"**不是同一命题**，**不能**揉进"loss of heterogeneity → 致痫"机制链（否则逻辑反复：一会儿窄危险、一会儿宽危险）。它是一条**独立**线索，价值在于"低门槛尾 = 成核风险"。这也是主实验局部化（外围安静）的动机。

> **证据等级纪律（2026-06-09 review）**：目前只能写成"**发现一个风险/强烈警报：宽参差的低门槛尾巴细胞可能触发整片自发爆发**"，**不能写成已坐实的稳定机制链**。它现在是**单次实现**（一个网络、一个随机尾部），有可能是"这个网络规模 + 这次随机尾部组合"下的单实现现象。要升级成机制结论，至少还需：**多种子**（尾部是否每次都触发）、**有限尺寸扫描**（是不是 finite-size 假象）、**密度扫描**（换密度还在不在）。在做完这些之前，措辞停在"风险线索"。

## 4. 与强制链 / 红线的对账

- **方向从计算出、允许 disconfirm（但措辞收紧）**：文献 Rich"异质性↓→更易同步"在**这个间期工作点、这个基质/尺度、std 1.5→0.5 局部核**这一组具体条件下**没有复现一个稳定方向的效应**（matched 跨位置无一致方向，见 §7）——这是单一工作点的 null，**不是对一个为不同基质/尺度提出的假设的证否**（§8 + 预注册 tier 纪律：写"没复现"不写"证否"）。**自点火（易激）的主门归到"平均更易兴奋"（mean shift）；参差宽窄只调点火可靠性（宽尾巴帮点火 84/84、收窄略降 77/84），不是零作用**——与 spec §7"非循环声明"一致。
- **中观参数层的诚实边界（回应用户"会影响哪些中观参数、仿真里有没有影响到"）**：SNN 这一轮直接量的是**有限尺寸可观测**——核内同步度、事件时移、传播轴角、是否自终止/自点着。spec §2 的中观读数（有效曲线斜率 G、曲率 H、跨单元离散 Var、闭环放大、安全余量 S）是**率均值场的干净抓手**，本轮**没有在 SNN 里重新测**：SNN 的价值正是率均值场抓不到的有限尺寸同步/成核，所以用有限尺寸 observable 回答，G/H/Var/S 留在率模型（Step3b 已算：参差↓ 动斜率 +13%、不动余量 S）。这是**有意分工**，不是把中观映射要求丢了。映射成立：底层旋钮（核内参差 / 均值）→ 有限尺寸可观测（同步/成核/自限）——参差这条 ≈ 无影响，均值那条 → 自点着。
- **配平核 ≠ 纯方差隔离（但 mean-only 轴已部分补上，review C/3B）**：matched 是门槛均值配平、非发放率 / F-I 斜率配平，仍带残留 Jensen。本轮加的 **mean-only 轴**（std 固定、mean 下移）让我们能把"平均"和"参差"两条轴分开问。结论分两层、**不能塌成"纯 mean"**：(a) **点火的主门是平均下移**；(b) **参差宽窄调节点火可靠性**（宽 84/84 vs 窄 77/84，宽尾巴帮点火，**不是零作用**）；(c) **点着后量级是否有 mean×variance 交互 = 仍没看清**（被核位置主导）。写"纯 mean"会抹掉 (b) 的参差作用、并把 (c) 的"没看清"误当成"无交互"。完整"纯方差轴"还需率配平 / F-I 斜率配平（roadmap 优先级 4）。
- **异质性 = 阈值离散度的窄 projection（review 2A）**：本轮把 Rich 宽义异质性 operationalize 成"局部点火门槛离散度"，**不**声称测了全部致痫异质性（突触/抑制/通道/架构）。措辞锁"operationalized heterogeneity as local spike-threshold dispersion"。
- **不配平核 = 局部更易激，不是细胞死亡**：unmatched 是 mean down（hyperexcitable），细胞丢失（密度↓/连接缺失）本轮不做。
- **gap-limit 未放宽**：std 仍锁 1.5/0.5；均值下移 2mV（间隙 7mV 的 ~28%）。

## 5. 诚实 caveat

- **种子不是完全独立的网络种子（关键 caveat，2026-06-09 review）**：sweep 的网络（神经元位置 + 连接结构）在 `_build(seeds[0])` 里**只建一次、固定在 seed 1**；12 个种子只重抽**阈值场实现 + 噪声/poisson**（kick 每个位置固定）。所以**连接骨架带来的变异没被采样**——这支持"在**同一张网络骨架**上跨位置 + 跨随机实现（阈值场/噪声）的机制筛查"，**还不够叫正式统计裁决**。下一步必须把**连接种子 / 阈值场种子 / 噪声-kick 种子**分开扫（见 §8）。grid 那 3 个 mid 种子同理（单网络）。
- 这一切仍是**机制筛查（mechanism screen），不是论文级统计**（需独立连接种子 + 位置/幅度 sweep）。
- 小尺度（L=3mm、电极按比例缩小、firing-density 读出**非 LFP**、非真实 SEEG 间距）——对应 SEEG 需 forward LFP/HFO 层。
- 自点火核效应强位置依赖，未系统扫位置；且其 d_core_paf 是自点火状态指标、不是诱发同步，两类指标未在 trial 级完全拆分（runner 已打 `evoked_clean` 标，统计拆分留 roadmap）。
- 机制图局部离散度上色：mean-shifted 核边界有平均跳变伪环（非核内真实参差，看核内部）。
- 自限指标已换干净版（rest[20,50]+衰减比，旧 [50,150] 弱窗已弃，review 1A）。**尾窗完整性保护（2026-06-09 review）**：`self_limit` 现在多返回 `tail_complete`——衰减窗 [peak+120, +200] 是否完整落在轨迹内；**晚峰**事件若衰减窗被仿真终点（T=450ms）截短，`returned` 只是弱证据。本轮 grid+sweep 事件峰 ~176ms（诱发）或戳前 ~70ms（自点火），衰减窗 ≤376ms，**都在 450ms 内、tail_complete=True**，`returned` 数可信；这个标主要为下一步"平均下移幅度扫"（更低均值可能推后峰）兜底。
- 引擎改动（kick_probe.py V_th_per_neuron）gitignored，已 patch + checksum + runner 断言追踪；每个结果 JSON 带 full provenance（git hash + 引擎 checksum + 配置 + 种子 + 度量版本，review 4B）。

## 7. 多种子 + 位置 sweep（2026-06-09，roadmap 优先级 1 第一档）

把 3 种子单位置的 grid 扩成**跨位置 + 多种子**，回答 grid 留下的两个问题：(A) 配平核（纯方差）的"无效应"扛不扛得住跨位置 + 多种子？(B) 参差到底调不调节"点着之后烧多旺"？结果 `results/topic4_sef_hfo/snn_heterogeneity/sweep_metrics.json`，图 `figures/sweep_ignition.png` + `sweep_matched_evoked.png`。

**规模（账要算清）**：固定戳点在长轴末端；把病理核搬到 **7 个位置**（5 个沿轴 far→near + 2 个离轴），每个位置 **12 个随机种子** → **7 × 12 = 84 个"位置-种子格"**。每格里又跑 baseline + 3 种病理改法（matched/mean_only/unmatched）配对 → **84 × 3 = 252 条病理-条件结果**。（早先口头汇报误写"84 = 7×12×3"，是错的；84 是位置-种子格、252 才是条件结果。）**重要 caveat（见 §5）**：12 个种子是在**同一张固定网络骨架**上重抽阈值场 + 噪声，**不是独立连接种子**——这是"同一网络上的跨位置/随机实现筛查"，不是正式统计裁决。

**(1) 自点火 = 平均为主门、参差调可靠性（headline，现在很硬）**：
- matched（纯方差，平均不动）：**84/84 一次都不点着**（跨全部位置 × 种子）。
- mean_only（宽 + 压低平均）：**84/84 全点着**。
- unmatched（窄 + 压低平均）：**77/84 点着**，偶尔（7 次）点不着。
- → 平均门槛下移是自点火的**主门**；**宽参差的低门槛尾巴让点火更保险（84/84），收窄略降到 77/84**——参差不是主开关、但调"点火可靠性"，方向与 B 侧"尾驱动成核"一致。**早先"参差零作用"的说法在此修正。**（图 `sweep_ignition.png` 直接画三条线 + 0/84、84/84、77/84 计数。）

**(2) 纯方差对诱发同步 = 跨位置无一致方向（不是"稳健零"）**：matched 在 7 个位置上的诱发同步改变量**正负都有、随位置摆动**（−0.18 @axis−0.3 → +0.22 @axis+0.3，沿轴还有个 far 负、near 正的位置梯度），**pooled mean −0.023±0.026（n=84）≈ 0**。诚实读法：**跨位置没有稳定同向效应**，而不是"零效应"——位置依赖的摆动本身要写出来（图 `sweep_matched_evoked.png` 标题就写"no consistent direction"）。这把 grid 的 −0.01±0.05 坐实并展开成位置谱。

**(3) "点着之后烧多旺"被位置主导、分不出参差作用（问题 B 仍未解）**：点着后的核内"烧多旺"指标随**核位置**大幅摆动（−0.28 → +0.49），mean_only（宽）与 unmatched（窄）两条线基本**重叠**。→ **位置才是主导变量，看不出参差对"点着后量级"的可分离作用**。注意：这个指标是**自点火状态强弱**、不是"被戳出来的同一个事件"的同步——按降级纪律（§2），自点火格子的这些 post-kick 数**只作状态指标，不和 matched 的诱发同步放同一条结论里比**。

**口径小结（本轮可保留 / 不可写）**：
- ✅ 可保留：「单独收窄核内门槛参差，在当前小尺度 / 工作点 / 0.5mm 核 / std 1.5→0.5 下，没有改变诱发传播事件的稳定方向」；「把核平均门槛下移到 16mV 后，核大多变成戳前自点着的局部起火点」；「本轮把'失去异质性'和'平均更易兴奋'分开讲」（最有价值）；「读出是 firing-density / 小尺度 / 缩放电极，还不能对应真实 SEEG」。
- ❌ 不可写：「点火完全由平均决定、参差零作用」（被 84/84 vs 77/84 否掉）；「纯方差是稳健零」（应写"跨位置无一致方向"）；把自点火后的"烧多旺"和 matched 的诱发同步混在一句结论里。
- 这仍是**机制筛查（mechanism screen）结果，不是 Step 3 正式结论**——升级需下面的最小补强。

## 8. 下一步候选（不在本轮）
