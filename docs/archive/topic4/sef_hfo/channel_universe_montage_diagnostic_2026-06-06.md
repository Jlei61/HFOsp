## 朴素话摘要（测了什么 / 怎么测的 / 揭示了什么）

**测了什么**：我们想在同一批病人身上，比"HFO 响多少次（发放率）"和"HFO 传播时谁先谁后的起止端（几何端点）"哪个更能稳定指出致痫区。要公平比，两个读数得落在同一根电极通道上。我去核对这两个读数到底在不在同一套通道上。

**怎么测的**：把三个来源（临床致痫区名单 / 每通道发放率 / 传播几何）的通道名逐被试对齐，看能对上多少；再看几何"够不着"的那些致痫区通道，它们的发放率排在本病人的第几名。

**揭示了什么**：
1. epilepsiae 的 15 个病人三套通道名完全一致，几何几乎覆盖全部致痫区（覆盖率中位 1.00）——干净。
2. yuquan 有两个问题：(a) 一部分病人的"发放率"用的是**双极对**命名（`E11-E12`），而几何/致痫区用**单触点**命名（`E11`）；(b) 即使名字对上了，**几何只覆盖约一半的临床致痫区**（覆盖率中位 0.52）。
3. 几何"够不着"的那些致痫区通道，**恰恰是低发放率的**（中位排在第 32 百分位，而几何能看到的致痫区通道排在第 76 百分位）。这说明：几何天生只活在"放电活跃区"里（要排先后顺序就得参与足够多的群体事件 = 高发放率）；而它够不着的低率致痫区，**单纯看发放率也照样够不着**——这是两种方法的**共同盲区**，不是几何独有的短板。
4. 双极命名问题可以**精确**解决（不是近似）：yuquan 的传播几何本来就来自双极检测，lagPat 把双极对 `E11-E12` 按"第一个触点"存成了 `E11`；这个约定在 8 个双极病人里逐一验证成立（几何触点都是某双极对的第一触点，且不是任何双极对的第二触点）。所以 `E11` → 唯一双极对 `E11-E12`，一一对应。

**对核心主张的影响**：原计划隐含"几何 vs 率"是两个独立定位器的平等对决；但几何**寄生在**率已经挑出的高率区里。所以诚实的主张要**收窄**为——

> **在两者都够得着的放电活跃区内，传播顺序（谁先谁后）是比发放计数（响多少次）更耐采样时间漂移的致痫区指纹。**

而**不是**"几何能定位率够不着的致痫区"（那部分是低率区，率和几何都瞎，补做检测也救不回——通道太安静、排不出稳定顺序）。这个收窄后的主张仍然支持"率会漂、不够"这条核心 argue，只是诚实划定了它成立的地盘。

## 2. first-contact 蒙太奇桥接（精确、确定性、已验证）

**约定**：lagPat 单触点标签 `X` = 双极对 `X-<next>` 的第一触点。

**验证**（8 个双极 yuquan 逐一）：几何触点集 ⊆ {双极对第一触点} **且** ⊄ {双极对第二触点}（如 `K3` 不是任何 `?-K3` 对的第二触点 → 排除"第二触点约定"）。7/8 完全唯一映射；litengsheng 3 个触点（A9/A10/B9）例外：A9/A10 在率文件中**没有任何双极对**（几何看到但率未检测）→ 标 missing；B9 只在 `B8-B9`（末端触点）→ 为保持桥接纯粹（不引入相邻对近似），也标 missing。

**锁定规则**：
- single-montage（6 yuquan + 15 epi）：几何触点 `X` → 精确同名率通道 `X`。
- bipolar-montage（8 yuquan）：几何触点 `X` → **唯一**双极对 `X-next` 的率；无唯一对 → **missing**（排除出 U，**不填 0**）。
- 门：映射到的率通道 `n_events ≥ 30`；被试 `total_hours ≥ 12`。

## 3. 根本发现：几何被高率约束 + 漏掉的 SOZ 是低率（共同盲区）

把每个 montage-clean 被试（15 epi + 6 single yuquan，n=21）的致痫区通道分三类，看发放率百分位：

| 类别 | 中位发放率百分位 | 含义 |
|---|---|---|
| 几何**覆盖**的 SOZ 通道 | **76** | 高率，几何与率都够得着 |
| 检测到但几何**够不着**的 SOZ 通道 (n=22) | **32** | 低率，几何排不出稳定顺序 |
| 未检测到的 SOZ 通道 | <检测下限 | 太安静，率与几何都看不到 |

逐被试一致（漏掉的 SOZ 永远比覆盖的低率）。**判别结论**：几何漏掉的致痫区是**低率**（不是"高率但被 refine 丢掉的"），所以补做非 lagPat 区域的群体事件监测**救不回**——低率通道参与事件太少，排不出稳定顺序。这片低率区是率与几何的**共同盲区**，是所有 HFO-based 定位的共同上限，不是几何独有缺陷。

**临床致痫区的嵌套结构**（yuquan 弥漫型尤其明显）：
`临床 SOZ ⊇ HFO-检测到的 ⊇ 几何可定义的（高率、可排序）`。
gaolan 是极端例子：几何活跃在 B/C/D 杆，临床 SOZ 在 A/B'/E/F 杆，几乎不相交（覆盖率 0）→ comparison-A 判 insufficient。

## 4. 收窄后的核心主张（locked）

> **放电活跃区内，传播顺序比发放计数更耐采样时间漂移的致痫区指纹。**

- ✅ 可证（不需补任何检测）：活跃区内 count vs order 谁更随采样时间稳定（同一批通道、同一批事件的两个正交读数，不循环）。
- ❌ 不可证（补检测也救不回）：几何定位率够不着的（低率）SOZ。
- 数据集分工：**epilepsiae 为主队列**（长程多日 117–435h = 采样时长脊梁的理想数据；覆盖率≈1.00 使"活跃区内主张"≈"完整定位主张"）；**yuquan 为补充层**，诚实暴露"临床 SOZ 约一半落在低率共同盲区"。

## 7. 几何分数定义（3 个分辨率层，locked）

用户列的三种端点定义对应三个分辨率层（跨 PR 合同已锁）：
- **PR-2 反向节点** = 整模板配对层，非通道级 → 仅作溯源/漏斗，**不当定位分数**。
- **固定模板前 n 端点** = 端点摘要层（固定 k 抽象）→ 离散敏感性（n=1,2,3）。
- **rank-displacement 源/汇互换节点** = 通道级最稳来源（变 k + 逐被试 null）→ 第三种几何口径。

**决定**：主用计划 §3.1 **连续两端端点分数**（最源/最汇→1、中间→0、对 A/B 平均，不依赖任意 n）；敏感性 = 离散前 n + 互换节点版。三者同在高率域，覆盖率限制相同；多测验证活跃区内排名稳健性，不改覆盖故事。
