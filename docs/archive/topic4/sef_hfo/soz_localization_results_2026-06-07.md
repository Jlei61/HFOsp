# SEF-HFO SOZ-localization — 结果（诚实负结果，主张未获支持）

**日期**：2026-06-07
**前置**：`channel_universe_montage_diagnostic_2026-06-06.md`（数据契约 + 主张收窄 v4）。
**结果文件**：`results/topic4_sef_hfo/soz_localization/{comparison_a.json, reliability.json, aligned_source_test.json}`。
**一句话**：在本数据上，"放电活跃区(SOZ∩U)内部，传播顺序比发放计数更耐采样时间漂移"这条核心主张**未获支持**——率在 SOZ 内部本身就很稳，几何各读数都没赢它。

---

## 朴素话摘要（测了什么 / 怎么测的 / 揭示了什么）

**测了什么**：在每个病人"既是致痫区、又有足够 HFO 活动"的那批通道里，问两件事——(A) 静态：能不能把致痫区通道挑出来；(B) 稳定性：录得短/事件少时，"挑哪几个通道当源头"这件事，用"响多少次(发放计数)"和用"传播谁先谁后(顺序)"哪个更不随采样窗口变。

**怎么测的**：(A) 在活跃区内算 ROC-AUC（致痫区 vs 非致痫区）。(B) 全程数据定一个"靶"（计数最高的几个 / 最靠源头的几个），再用短窗/固定事件数重算，看短窗靶和全程靶的重叠(Jaccard)与排序相关(Spearman)；并用"把全程事件打散再随机抽同样多个"当对照零分布，区分"事件少导致的抖"和"真实时间漂移"。

**揭示了什么**：
1. **静态定位(A)在主队列基本无意义**：焦点型病人活跃区≈致痫区，活跃区里几乎没有非致痫区通道作对照，AUC≈0.5 是结构性的（5 个病人活跃区全在致痫区内）。这不是发现，是说明"主队列不适合做粗粒度致痫区二分类"。
2. **稳定性(B)：率在致痫区内部很稳，几何没赢**：
   - 率(在群体传播事件里的参与计数)的靶随采样很稳定(中位 Jaccard 0.60→0.78 随预算增大)。
   - "把所有事件混着算的传播顺序(source/sink/端点)"都**没比率稳**；混着算的 source **显著比率更不稳**（p(率>source)≈0.01–0.03），端点≈打平。
   - 怀疑混着算把正反两套模板的源头/终点平均糊掉了，于是按模板分开重算 source——但**诚实的分模板检验也不显著**：用全程模板对齐(rank_a)挑出"源头模板"再测，主队列 epilepsiae M=100 时 source 比率高一点点(Δ+0.14)但 **p=0.20 不显著**，M=200 边缘(p=0.08)，yuquan 平。事后取 fwd/rev 里更好那个会显著(p=0.01)，但那是**赢家诅咒偏差**，不算数。
3. **最重要的反证(reframe)**：率在 SOZ **内部**不一定漂。之前"率会漂"的现象，很可能主要发生在**更大尺度**（哪些通道算活跃、跨活跃区边界的成员变化），**不是** SOZ∩U 内部的排序。所以"用几何替代率做 SOZ 内部精细定位"这条路，在本数据上没走通。

## 护栏 / 局限（必读）

- **多重比较 / 花园岔路**：本分析跑了 ~5 种几何读数 × 4 个事件预算 × 2 个指标 × 3 个队列。**没有任何 p 是确证性的**，全部只能作"产生假设"，要确证需预注册 + 独立 held-out 队列。
- **分模板的事件预算不公平**：当前预算 M 是总事件数，分到每个模板的事件数不固定（对齐 source 实际只用了约 M/2 个事件，反而吃亏）。严肃的模板内 source 稳定性检验应"每个模板各给自己的事件预算"再比——这是未来工作，不是本轮结论。
- **赢家诅咒**：`reliability.json` 里 `best_stratified_source_jaccard`（每被试事后取 fwd/rev 更优者）显著但**偏差向上**，已在 meta 标注 `do_not_report`；决定性结论以 `aligned_source_test.json` 为准。
- **rate 的语义**：这里"率"=群体传播事件里的通道参与计数（与几何同一批事件，公平），**不是泛义 HFO rate**；文章语言须如实写。
- **覆盖率/盲区**：yuquan 几何 SOZ 覆盖率中位 0.52；低率 SOZ 是率与几何共同盲区（见诊断 doc §3）。

## 对文章的意义

两个预注册分析（静态定位 + 采样稳定性）都对"几何在 SOZ 内部胜过率"给出**负/弱**结果。最稳的诚实陈述不是"几何更稳"，而是：**"粗粒度 SOZ 二分类在焦点型队列里无对照度；真正该问的是活跃 SOZ 内部的时间可靠性——本数据显示率在 SOZ 内部已相当稳定，传播顺序未显著更优；率的时间漂移更可能是跨活跃区边界的成员现象，而非 SOZ 内部排序。"** 文章方向需 user 重定（不得按"几何更稳"包装）。
